# 카드 에디터 표/YouTube 표현력 확장 2026-05-18

## 2차

### 작업내용

- 1차 복합 클립보드 안정화 위에 카드 에디터의 표 붙여넣기/표 삽입 기능을 추가한다.
- YouTube URL 단독 붙여넣기와 렌더링 시 standalone YouTube 링크를 안전한 embed로 전환한다.
- Yeon 카드 에디터의 흰 배경 작성/미리보기 디자인, 툴바 톤, 카드 프리뷰 구조는 유지한다.

### 논의 필요

- Mermaid/코드블록 하이라이팅은 이번 차수에서 제외한다.
- YouTube iframe은 `youtube-nocookie.com` 기반으로 정규화하고, 허용되지 않는 iframe은 제거한다.

### 선택지

1. 표와 YouTube를 모두 에디터 입력 단계에서 TipTap 노드로 저장한다.
2. 표는 Markdown table 문자열로 저장하고, YouTube는 안전한 iframe HTML 노드로 저장/렌더링한다.
3. 렌더링 단계에서만 변환하고 에디터 입력은 건드리지 않는다.

### 추천

- 2번. 기존 HTML 기반 카드 에디터 저장 흐름과 가장 잘 맞고, 사용자 입력 즉시 미리보기 일관성을 확보할 수 있다.

### 사용자 방향

- 이전 계획의 다음 차수를 진행한다. 디자인은 유지하고 가져올 기능만 선별한다.
