# Todo 집중 화면 MoodDesk 감성 복구

## 1차

### 작업내용

- Todo 생성 영역의 우선순위 드롭다운을 제거한다.
- 우선순위/예상 시간 버튼은 필터가 아니라 새 할 일 생성 기본값으로만 동작하게 정리한다.
- Today 카드에 명시적인 시작 버튼을 추가한다.
- 아이콘 버튼에는 마우스 hover/focus 시 설명 툴팁을 노출한다.
- 시작 버튼은 Todo 집중 화면으로 이동하고, 집중 화면에서 작업 완료 시 해당 일을 완료 처리한 뒤 Todo 보드로 돌아온다.
- 집중 화면은 삭제 전 MoodDesk/마음저널 화면의 영상 자산과 차분한 공간감을 Todo 작업 화면으로 복구한다.

### 논의 필요

- 삭제된 card-service `/study-desk` 제품 표면은 되살리지 않는다.
- 복구 대상은 Todo의 새 집중 화면에서 쓰는 MoodDesk 화면감과 자산으로 한정한다.

### 선택지

- A. 기존 MoodDesk 정적 페이지를 그대로 복구하고 Todo에서 링크한다.
- B. MoodDesk 자산을 복구하되 Todo 전용 React 화면으로 재구성한다.

### 추천

- B. 이미 제거한 card-service 작업대 제품 표면을 다시 노출하지 않고, 사용자가 요구한 Todo 시작 흐름에만 집중 화면을 붙일 수 있다.

### 사용자 방향

- 추천 기준으로 진행한다.
