# 타자 점령전 로비 색상 토큰 정리 백로그

## 1차: 점령전 대기방 임의 색상 제거

- 작업내용
  - 점령전 대기방 UI에서 sky/red/pink/green 등 임의 Tailwind 색상을 제거한다.
  - 현재 디자인 시스템의 흰 배경 타자 서비스 정규 색상과 타이핑 오렌지 강조만 사용한다.
  - React Native/Universal UI 전환 계획에 맞게 새 색상 의미를 웹 화면 안에서 임의로 만들지 않는다.
- 논의 필요
  - 팀 구분 색상은 향후 `@yeon/design-tokens`에 semantic token으로 등록한 뒤 다시 적용할지 여부.
- 선택지
  - A. 현재는 중립 토큰 기반으로 정리하고, 팀 색상은 텍스트/구조로만 구분한다.
  - B. 웹에서만 임시 팀 색상을 유지한다.
- 추천
  - A. RN 공유 토큰 계획과 충돌하지 않게 임시 색상 사용을 중단한다.
- 사용자 방향
  - 색을 임의로 쓰지 않고 사용하는 디자인 토큰만 사용한다.
