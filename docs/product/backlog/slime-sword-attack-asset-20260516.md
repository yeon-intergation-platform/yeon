# 슬라임 검 공격 에셋 개선 백로그 (2026-05-16)

## 1차

### 작업내용

- `/slime-game` 공격이 팔을 뒤로 휘두르는 것처럼 보이는 문제를 개선한다.
- 새 투명 SVG 검 베기 에셋을 추가한다.
- attack action 중 슬라임 앞쪽에 검 궤적을 오버레이해 J 키 입력이 검으로 휘두르는 공격처럼 보이게 한다.
- 전투 판정, 히트박스, 데미지, 몬스터 연결은 추가하지 않는다.

### 논의 필요

- 현재 슬라임 sheet의 attack 프레임 자체는 팔 움직임이 약하므로 원본 sheet를 다시 그리는 대신 별도 검/슬래시 에셋을 합성한다.
- 검 에셋은 게임 판정 source of truth가 아니라 attack action의 시각 피드백이다.

### 선택지

- A. `sword_slash_attack.svg`를 새 에셋으로 만들고 attack 상태에서 stage에 합성한다.
- B. 기존 slime sprite sheet 자체를 다시 제작한다.
- C. CSS 도형만으로 검을 그린다.

### 추천

- A. 새 검 에셋 요구를 만족하면서도 슬라임 sheet와 action 도메인을 크게 흔들지 않는다.

### 사용자 방향

- A 기준으로 진행한다.
