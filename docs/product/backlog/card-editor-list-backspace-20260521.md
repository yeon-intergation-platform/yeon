# 카드 에디터 빈 목록 항목 Backspace 동작 보정 (2026-05-21)

## 1차

### 작업내용

- 카드 추가/수정 리치 에디터에서 빈 목록 항목에 커서가 있을 때 Backspace를 누르면 이전 항목 끝으로 병합되는 동작을 막는다.
- 빈 목록 항목은 목록 밖 일반 문단으로 전환하고, 커서는 기존 아래 줄 위치에 유지되게 한다.
- 불릿 목록과 숫자 목록 모두 같은 규칙으로 처리한다.

### 논의 필요

- 빈 목록 항목에서 Backspace를 기존 에디터 기본 병합 동작으로 둘지, 목록 종료 동작으로 바꿀지.

### 선택지

1. 기본 병합 유지: 브라우저/ProseMirror 기본 동작과 같지만 목록 뒤 일반 문단 작성이 막힌다.
2. 목록 종료로 보정: 빈 항목을 일반 문단으로 lift해 커서를 아래에 유지한다.

### 추천

- 2안. 사용자가 목록 뒤 일반 문단을 작성할 수 있어야 하므로 빈 목록 항목 Backspace는 목록 종료 동작이 맞다.

### 사용자 방향

- 현재처럼 커서가 위로 올라가면 안 된다. 줄바꿈은 지워지되 커서는 아래에 존재해야 한다.
