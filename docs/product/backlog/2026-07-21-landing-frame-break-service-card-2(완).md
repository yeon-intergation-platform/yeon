# 랜딩 프레임 브레이크 카드 전등 돌출 보완

> 상태: 완료

## 1차 — 전등 프레임 브레이크와 헤더 안전영역

- 작업내용: 타자연습 카드의 투명 전경에서 인물뿐 아니라 전등도 이미지와 헤더 경계를 넘도록 노출하고, 제목이 전등과 겹치지 않도록 반응형 헤더 여백을 조정한다.
- 논의 필요: 375px 모바일과 3열 데스크톱 카드 모두에서 전등 돌출을 유지하면서 제목·상태 배지의 가독성을 보장해야 한다.
- 선택지:
  - A. 기존 전경을 좌·우 마스크로 나눠 전등과 인물을 각각 돌출하고 헤더 높이·패딩으로 세로 안전영역을 만든다.
  - B. 전등만 별도 이미지 에셋으로 다시 분리한다.
- 추천: A. 기존 에셋을 재사용해 화질과 정렬을 유지하고 변경 범위를 CSS 레이어에 한정한다.
- 사용자 방향: 전등도 돌출하되 제목과 겹치지 않도록 패딩을 추가한다.

## 완료 조건

- 전등과 인물 모두 카드 이미지 상단 경계를 넘어 보인다.
- 제목, 번호 배지, 운영 상태가 전등·인물과 시각적으로 겹치지 않는다.
- 375px 및 1440px에서 가로 스크롤 없이 카드 레이아웃이 유지된다.
- Playwright 스크린샷과 lint/typecheck 결과를 작업 로그에 남긴다.

## 결과

- 헤더를 96px로 확장하고 제목 행 아래에 32px 패딩을 확보했다.
- 동일 전경을 이미지 영역과 헤더 돌출 영역에 맞춰 정렬해 전등 갓과 인물 머리가 함께 경계를 넘도록 했다.
- 375px 및 1440px에서 제목·번호·운영 상태가 전경과 겹치지 않고 가로 스크롤이 없음을 확인했다.
