# 카드 덱 상세 카드 추가 버튼 위치 정리

## 1차

- 작업내용: 카드 덱 상세 화면의 `+ 카드 추가` 버튼을 페이지 최상단 단독 위치에서 카드 목록 헤더 액션으로 이동하고, 덱 헤더에 남아 있는 `YEON` 외부 링크 버튼을 제거한다. 모바일에서는 카드 목록 수량과 추가 버튼이 같은 액션 줄에 보이도록 정리한다.
- 논의 필요: 카드 추가를 상단 고정 CTA로 유지할지, 카드 목록 소속 액션으로 둘지 여부.
- 선택지:
  - A. 카드 목록 헤더 오른쪽에 `+ 카드 추가`를 둔다.
  - B. 덱 제목 영역의 작업 버튼 그룹에 둔다.
  - C. 화면 하단 floating action button으로 둔다.
- 추천: A. 카드 추가는 카드 목록을 조작하는 액션이므로 목록 헤더에 소속시키는 것이 데스크톱/모바일 모두 자연스럽다.
- 사용자 방향: A. 현재 최상단 위치가 이상하고 모바일에서도 이상하므로 위치를 바로잡고, `YEON` 버튼은 제거한다.
