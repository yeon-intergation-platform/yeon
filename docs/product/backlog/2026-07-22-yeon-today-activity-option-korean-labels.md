# YEON Today 활동 옵션 한국어화 백로그

## 목표

하루 기록의 활동 관리 화면에서 서버 계약용 색상·아이콘 코드가 영어로 노출되지 않도록 한국어 표시 라벨을 적용한다. 저장값과 API 계약은 기존 영문 토큰을 유지한다.

## 1차 — 표시 라벨 분리

- 작업내용: 활동 색상과 아이콘의 계약값별 한국어 라벨을 단일 매핑으로 정의하고, 두 선택 상자가 해당 라벨을 표시하도록 변경한다.
- 논의 필요: `circle` 아이콘을 사용자에게 어떤 용어로 보여줄지 결정해야 한다.
- 선택지: `원형`, `점`, `기타 아이콘`.
- 추천: 화면 정의서의 기본 활동 아이콘 명칭과 일치하도록 `점`으로 표시한다.
- 사용자 방향: 한국어로 표시한다. 저장값과 API 계약값은 변경하지 않는다.

## 2차 — 회귀 검증과 시각 확인

- 작업내용: 린트·타입 검사·웹 테스트를 실행하고, 실제 활동 관리 화면에서 모든 색상·아이콘 옵션이 한국어인지 확인한다.
- 논의 필요: 없음.
- 선택지: DOM 옵션 텍스트 검증만 수행, DOM 검증과 실제 화면 캡처를 함께 수행.
- 추천: DOM 검증과 실제 화면 캡처를 함께 수행한다.
- 사용자 방향: 추천 기준으로 진행한다.

## 완료 조건

- 색상 옵션은 `파랑`, `초록`, `주황`, `보라`, `노랑`, `빨강`, `회색`으로 표시된다.
- 아이콘 옵션은 `책`, `게임패드`, `식기`, `자동차`, `컵`, `달`, `운동`, `점`으로 표시된다.
- 생성 요청에는 기존 영문 토큰이 전송되고 서버 저장 데이터와 호환된다.
- 관련 정적 검사와 브라우저 확인이 통과한다.
