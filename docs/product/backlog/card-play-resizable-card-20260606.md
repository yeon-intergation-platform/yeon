# 카드 플레이 카드 크기 조절 백로그 (2026-06-06)

## 1차수

### 작업내용
- 플래시카드 플레이 화면의 카드 컨테이너를 우하단 핸들로 가로/세로 조절할 수 있게 한다.
- 사용자가 조절한 크기는 같은 덱의 다음/이전 문제로 이동해도 유지한다.
- 새로고침 후에도 같은 덱에서 유지되도록 브라우저 로컬 저장소에 저장한다.

### 논의 필요
- 크기 저장 범위를 현재 브라우저/덱 단위로 둘지, 계정 서버 설정으로 확장할지.

### 선택지
1. 클라이언트 로컬 저장소에 덱별 크기를 저장한다.
2. URL 쿼리에 크기를 저장한다.
3. 서버 사용자 설정 API를 추가한다.

### 추천
- 1번. 신규 백엔드 ownership 없이 카드 플레이 UX만 개선하고, 문제 번호 이동과 새로고침까지 자연스럽게 유지한다.

### 사용자 방향
- 추천 기준으로 진행한다.
