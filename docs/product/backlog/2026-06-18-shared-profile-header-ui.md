# 공통 프로필/헤더 UI 정리

## 1차

### 작업내용

- `typing.yeon.world`와 `card.yeon.world` 홈에서 공통 프로필 카드의 캐릭터 선택 버튼 텍스트가 사라지는 문제를 재현하고 수정한다.
- 타자 홈 레이아웃 폭과 패널 규격을 카드 홈 기준으로 맞춘다.
- 공통 제품 헤더의 프로필 메뉴 간격과 설정 버튼 아이콘 품질을 정리한다.
- `yeon.world` 서비스 목록에서 `news.yeon.world`로 이동할 수 있게 서비스 레지스트리에 뉴스 진입점을 추가한다.

### 논의 필요

- 뉴스/블로그/support 전체를 한 번에 노출할지 여부.

### 선택지

- A. 이번 작업에서는 사용자가 명시한 `news.yeon.world`만 홈 서비스 카드에 추가한다.
- B. `news`, `blog`, `support`를 모두 추가한다.

### 추천

- A. 현재 요청의 직접 대상은 news 진입점이며, blog/support는 콘텐츠/라우팅 완성도 검증 후 별도 노출하는 편이 안전하다.

### 사용자 방향

- 추천 기준으로 진행한다.
