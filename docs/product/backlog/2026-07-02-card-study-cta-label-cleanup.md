# 카드 학습 CTA 명칭 정리 백로그

## 목표

`card-service` 덱 상세에서 Study Desk 제거 이후 남은 기본 학습 CTA를 `학습하기`로 정리한다.

## 1차: 덱 상세 학습 진입 정리

### 작업내용

- 최신 main 기준 `/card-service/study-desk` 라우트와 `25분 집중 작업대` CTA가 코드에서 제거되었는지 확인한다.
- 덱 상세의 기본 학습 CTA 문구를 `학습하기`로 변경한다.
- 카드 학습 경로 자체는 기존 `/card-service/decks/[deckId]/play` 흐름을 유지한다.

### 논의 필요

- 카드방 등 다른 화면의 `학습 시작` 문구까지 한 번에 바꿀지 여부.

### 선택지

- A. 사용자가 지목한 덱 상세 CTA만 `학습하기`로 바꾼다.
- B. 카드 서비스 전체의 모든 `학습 시작` 문구를 일괄 변경한다.
- C. 문구 변경 없이 Study Desk 제거 상태만 확인한다.

### 추천

A를 추천한다. 요청한 URL의 덱 상세 CTA만 바꾸면 영향 범위가 작고, 카드방 같은 다른 학습 진입 UX를 불필요하게 건드리지 않는다.

### 사용자 방향

Study Desk 관련 코드와 작업대 버튼을 제거하고, `일반학습` 명칭을 `학습하기`로 변경한다.

## 완료 조건

- 코드 기준 `study-desk` 라우트와 `25분 집중 작업대` CTA가 없다.
- 덱 상세 기본 학습 CTA가 `학습하기`로 표시된다.
