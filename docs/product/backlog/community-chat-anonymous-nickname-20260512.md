# community chat anonymous nickname 2026-05-12

## 배경

실시간 채팅은 익명성이 핵심이므로 로그인 계정의 이름, 이메일 앞부분, 내부 사용자 ID가 채팅 닉네임처럼 노출되면 안 된다. 채팅 전송은 게스트 동작을 유지하되 표시명은 익명 표시명만 사용한다.

## 1차

### 작업내용

- `/api/v1/community-chat/messages` 전송에서 계정 세션 기반 표시명과 userId 전달을 제거한다.
- 실시간 채팅 목록에서 내 메시지도 계정/아이디 느낌의 별도 라벨로 바꾸지 않고 서버가 반환한 익명 표시명을 그대로 보여준다.
- 기존 채팅 폴링, 전송, 접속자 수 표시는 유지한다.

### 논의 필요

- 없음. 실시간 채팅 익명성 원칙을 우선해 계정 식별자 연결을 끊는다.

### 선택지

1. 모든 채팅 메시지를 `익명이` 같은 익명 표시명으로 통일한다.
2. 세션별 임의 별칭을 새로 만들되 계정 정보와 연결하지 않는다.

### 추천

- 1번. 세션별 별칭도 장기 식별자로 오해될 수 있으므로 이번 수정은 계정/아이디 노출 차단에 집중한다.

### 사용자 방향

- 추천 기준으로 진행한다.
