# 랜딩 Discord AI·게임 프레임브레이크 전경 적용

> 상태: 완료

## 1차 — 서비스별 투명 전경 제작

- 작업내용: 제공된 Discord AI 어시스턴트와 게임 이미지를 1·2·3번과 같은 와이드 투명 전경 에셋으로 제작한다.
- 논의 필요: 원본 전체 장면을 그대로 축소하면 전경이 배경을 덮고 프레임브레이크가 약해진다.
- 선택지:
  - A. 원본 이미지를 카드 배경으로 통째로 사용한다.
  - B. 인물·전등·핵심 서비스 소품만 낮은 책상선의 투명 전경으로 재구성한다.
- 추천: B. 공통 배경을 유지하면서 서비스별 행동과 헤더 돌출이 함께 보인다.
- 사용자 방향: 1·2·3번과 같은 방식으로 6번과 8번에 적용한다.

## 2차 — 카드 연결과 구도 보정

- 작업내용: 6번에는 Discord 패널·인물·전등, 8번에는 게임 화면·인물·전등 전경을 연결하고 카드별 위치를 맞춘다.
- 논의 필요: 작은 카드에서는 UI 패널과 모니터가 인물보다 강하게 보일 수 있다.
- 선택지:
  - A. 모든 원본 소품을 유지한다.
  - B. 서비스 정체성에 필요한 소품만 남기고 인물을 중앙에 둔다.
- 추천: B.
- 사용자 방향: 기존 카드 배경은 건드리지 않는다.

## 3차 — 반응형 검증

- 작업내용: 375px·1040px·1440px에서 제목 충돌, 전경 잘림, 공용 배경 노출을 확인한다.
- 논의 필요: 없음.
- 선택지:
  - A. 데스크톱만 확인한다.
  - B. 모바일·태블릿·데스크톱을 함께 확인한다.
- 추천: B.
- 사용자 방향: localhost에서 바로 확인한다.

## 완료 조건

- 6번과 8번에서 인물과 전등이 이미지 영역 상단을 자연스럽게 넘는다.
- Discord 패널과 게임 화면이 서비스 정체성을 분명히 보여준다.
- 1~8번이 동일한 공용 배경을 유지한다.
- 세 화면폭에서 제목 충돌과 가로 스크롤이 없다.

## 완료 결과

- 6번에 1921×786 Discord AI 투명 전경을 적용했다.
- 8번에 1762×766 게임 투명 전경을 적용했다.
- 두 카드만 전경 폭을 약 9% 확대하고 하단 안전 여백을 보정해 1·2·3번과 돌출 강도를 맞췄다.
- 공용 배경의 git blob 해시가 `origin/main`과 동일함을 확인했다.
- 375px·1040px·1440px에서 공용 배경 9개, 전경 8개, 가로 overflow 0을 확인했다.
