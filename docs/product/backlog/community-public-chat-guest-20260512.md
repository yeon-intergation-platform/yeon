# community-public-chat-guest-20260512

## 1차수

### 작업내용

- 커뮤니티 실시간 채팅이 chat-service 로그인/1:1 채팅방에 의존하는 구조를 제거한다.
- Spring을 공개 커뮤니티 채팅 메시지의 SSOT로 두고, Next.js는 BFF와 화면 연결만 담당한다.
- 비회원은 게스트 세션/닉네임으로 채팅할 수 있고, 루트 로그인 사용자도 chat-service OTP 로그인 없이 채팅할 수 있게 한다.

### 논의 필요

- 장기적으로 공개 채팅 신고/차단/관리자 삭제 정책은 별도 정책이 필요하다.

### 선택지

1. 기존 chat-service 1:1 채팅방 API를 게스트까지 확장한다.
2. 커뮤니티 전용 공개 채팅 API를 Spring에 추가하고 UI를 전환한다.

### 추천

- 2번. 1:1 채팅방 권한 모델과 공개 커뮤니티 채팅의 source of truth를 분리해 로그인 오류와 권한 혼선을 제거한다.

### 사용자 방향

- 추천 기준으로 진행한다.
