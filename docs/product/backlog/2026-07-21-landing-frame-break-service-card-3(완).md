# 랜딩 프레임 브레이크 전등 절단면 보정

> 상태: 완료

## 1차 — 전등 절단면 보정

- 작업내용: 원본 전경 상단에서 잘린 전등 목 부분이 프레임 브레이크 영역에 직선으로 드러나지 않도록 절단면 뒤에 전등 색상의 작은 둥근 캡을 연결한다.
- 논의 필요: 전등 갓과 관절의 돌출감은 유지하면서 제목 안전영역과 이미지 경계의 연결이 자연스러워야 한다.
- 선택지:
  - A. 절단면 위에 전등 색상과 같은 작은 둥근 캡 레이어를 연결한다.
  - B. 이미지 생성으로 전등의 잘린 윗부분을 새로 그린다.
- 추천: A. 기존 일러스트와 인물 외곽을 변경하지 않고 절단된 전등 목만 최소 범위로 연장한다.
- 사용자 방향: 잘려 보이는 전등을 더 자연스럽게 보정한다.

## 완료 조건

- 전등 목 부분의 수평 절단면이 직접 보이지 않는다.
- 전등 갓과 인물 머리의 프레임 브레이크 효과가 유지된다.
- 제목·번호·운영 상태와 전경이 겹치지 않는다.
- 375px 및 1440px에서 가로 스크롤과 이미지 로드 문제 없이 확인한다.

## 완료 결과

- 전등 목 절단면 뒤에 반응형 위치·폭의 둥근 캡을 추가해 평평한 상단을 자연스럽게 연장했다.
- 전등 갓, 인물, 제목 안전 여백과 기존 프레임 브레이크 크기는 유지했다.
- 375px 모바일과 1440px 데스크톱에서 가로 스크롤 부재와 이미지 로드를 확인했다.
