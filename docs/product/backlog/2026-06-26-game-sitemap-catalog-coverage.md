# 게임 sitemap 카탈로그 커버리지 보강

## 1차수

### 작업내용

- `game.yeon.world` host별 sitemap이 게임 카탈로그의 모든 slug를 포함하는지 테스트로 고정한다.
- 카탈로그에 새 게임이 추가될 때 Search Console 제출 대상 sitemap에서 빠지는 회귀를 줄인다.

### 논의 필요

- 없음. 기능 변경 없이 테스트 커버리지 보강만 수행한다.

### 선택지

- A. sitemap 생성 로직을 수정한다.
- B. 기존 SEO 테스트에 게임 카탈로그 전체 포함 검증을 추가한다.

### 추천

- B. 현재 생성 로직은 `getGameSlugs()` 기반이므로 로직 변경보다 회귀 테스트가 더 작고 안전하다.

### 사용자 방향

- 추천 기준으로 진행한다.
