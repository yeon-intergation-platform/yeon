# 서비스 소개 섹션 도움말 모달 이동 백로그

## 1차: 하단 소개 섹션을 제거하고 도움말 모달로 이동

- 작업내용
  - 타자·카드·커뮤니티 서비스 본문 하단에 직접 노출되던 소개/기능/FAQ 섹션을 제거한다.
  - 각 서비스 우측 상단 도움말 버튼에서 동일 성격의 안내 모달을 열도록 한다.
  - 서비스별 도움말 내용은 각 서비스의 기존 소개/기능/FAQ 데이터 또는 카테고리 설명을 재사용한다.
- 논의 필요
  - 검색 색인을 위해 숨김 SSR 텍스트를 별도로 둘지 여부.
- 선택지
  - A. 사용자 화면에서 어색한 하단 섹션을 완전히 제거하고 도움말 모달에만 둔다.
  - B. 하단 섹션은 시각적으로 숨기고 크롤러용으로 유지한다.
- 추천
  - A. 사용자가 지적한 문제는 서비스 화면 내 갑작스러운 섹션 노출이므로 사용자 화면에서 제거한다. JSON-LD/metadata는 유지해 검색 힌트는 남긴다.
- 사용자 방향
