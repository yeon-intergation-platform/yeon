# 랜딩 커뮤니티 카드 전경 v5 교체 (완료)

## 1차수 — 새 커뮤니티 전경 적용

### 작업내용

- 사용자가 제공한 커뮤니티 이미지를 투명 전경 에셋으로 누끼 처리한다.
- 4번 커뮤니티 카드의 기존 v4 전경을 새 v5 전경으로 교체한다.
- 1~9번 카드가 공유하는 `baekji-frame-break-background.webp`는 변경하지 않는다.
- 전등, 대화 말풍선, 공지 패널, 인물과 책상이 프레임 브레이크 전경으로 자연스럽게 보이도록 카드 구도를 조정한다.
- Playwright로 데스크톱과 모바일 화면을 캡처하고 오버플로·이미지 로딩을 확인한다.

### 논의 필요

- 없음. 사용자가 새 이미지를 커뮤니티 카드 전경으로 누끼 적용하도록 명시했다.

### 선택지

1. 기존 v4 파일을 덮어쓴다.
2. v5 파일을 추가하고 코드 참조를 v5로 변경한다.

### 추천

- 2번. 에셋 롤백과 변경 추적이 쉽고 브라우저 캐시 충돌을 줄일 수 있다.

### 사용자 방향

- 새 이미지로 커뮤니티 카드를 교체하고 누끼 전경으로 적용한다.

## 완료 결과

- `community-frame-break-foreground-v5.webp` 투명 전경을 추가하고 4번 커뮤니티 카드 참조를 교체했다.
- 공통 `baekji-frame-break-background.webp`의 Git blob hash가 `origin/main`과 동일함을 확인했다.
- Playwright 1440px, 1040px, 375px 화면에서 전등·대화 5개·공지·인물·책상 구도를 확인했다.
- 전체 가로 오버플로 0px, 전경 9개와 공통 배경 9개 로딩을 확인했다.
- lint, typecheck, parity, build, diff, skills sync, SSOT 검증을 통과했다.
