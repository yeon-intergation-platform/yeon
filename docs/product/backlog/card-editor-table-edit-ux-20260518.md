# 카드 에디터 표 편집 UX 강화 백로그 (2026-05-18)

## 배경

카드 에디터는 표 삽입과 표 붙여넣기를 지원하지만, 삽입 이후 Markdown table 라인을 직접 다뤄야 한다. 새 무거운 테이블 에디터를 도입하지 않고 기존 카드 에디터의 흰 배경·작은 툴바 디자인을 유지하면서 표 편집 보조 UX를 추가한다.

## 1차

### 작업내용

- 현재 커서가 Markdown table 안에 있을 때만 작은 표 편집 바를 표시한다.
- 표 행 추가, 열 추가, 표 정렬 기능을 제공한다.
- 표 삽입 기본 템플릿을 기존 2열 유지하되 편집 보조 기능과 호환되게 한다.
- 기존 HTML/TSV 표 붙여넣기 및 프리뷰 렌더링 동작은 유지한다.

### 논의 필요

- 셀 단위 WYSIWYG 편집까지 갈지 여부.
- 행/열 삭제 버튼까지 이번 범위에 포함할지 여부.

### 선택지

1. Markdown table 보조 바로 행/열 추가와 정렬만 제공한다.
2. TipTap table node 기반 WYSIWYG 테이블 에디터를 도입한다.
3. 표 편집은 유지하고 문서화/도움말만 추가한다.

### 추천

1번. 현재 저장 구조와 프리뷰 파이프라인을 유지하면서 사용자가 가장 자주 겪는 “행/열 추가가 불편함”을 해결한다.
