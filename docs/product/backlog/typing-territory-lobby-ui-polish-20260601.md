# 타자 점령전 로비/플레이 UI 보정 백로그

## 1차: 타자방 대기 화면을 점령전 로비처럼 보이게 보강

- 작업내용
  - 기존 타자방 대기 화면 상단에 레퍼런스형 점령전 로비 패널을 추가한다.
  - 좌측에는 방 정보(방 이름, 게임 타입, 게임 시간, 보너스 게임, 현재 인원), 이모티콘, 상세 로그를 배치한다.
  - 중앙/우측에는 주황팀/파랑팀 슬롯과 참가자 준비 상태, 방장 표시, 빈 슬롯/잠금 슬롯을 표시한다.
  - 하단에는 방 나가기, 팀 이동, 준비/준비 해제, 점령전 입장 액션을 큰 버튼으로 배치한다.
- 논의 필요
  - 점령전 전용 로비 route를 별도로 만들지 여부.
- 선택지
  - A. 기존 타자방 대기 화면에 점령전 로비 패널을 추가한다.
  - B. 점령전 전용 로비 route를 분리한다.
- 추천
  - A. 현재 사용자가 지적한 “로비에 참가한 다음 점령전” 흐름을 기존 타자방 참가 상태 위에서 바로 보강한다.
- 사용자 방향

## 2차: 점령전 플레이 화면을 레퍼런스 톤으로 보정

- 작업내용
  - 팀 라벨을 주황/파랑 계열로 통일하고, 보드 안내/하단 입력부가 카드 뒤집기 게임처럼 보이도록 문구와 카드 영역을 보정한다.
  - 현재 서버 프로토콜과 seed는 유지한다.
- 논의 필요
  - 카드 뒤집기 턴제 규칙까지 서버 프로토콜로 구현할지 여부.
- 선택지
  - A. 현재 단어 점령 규칙을 유지하고 시각/문구를 먼저 맞춘다.
  - B. 서버 프로토콜을 카드 뒤집기 턴제로 재설계한다.
- 추천
  - A. UI 불일치 해소가 우선이며, 서버 규칙 변경은 별도 차수로 둔다.
- 사용자 방향
