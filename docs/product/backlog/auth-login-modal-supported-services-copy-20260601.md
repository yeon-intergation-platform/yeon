# 로그인 모달 지원 서비스 문구 보정 백로그

## 1차: 동결 서비스명을 로그인 모달에서 제거

- 작업내용
  - 로그인 모달의 설명 문구에서 유지보수 동결 대상인 상담 워크스페이스 언급을 제거한다.
  - 현재 지원 대상인 카드·타자·커뮤니티 흐름에 맞는 공통 계정 안내로 바꾼다.
- 논의 필요
  - 루트 랜딩 전체의 상담 워크스페이스 마케팅 문구도 함께 정리할지 여부.
- 선택지
  - A. 사용자가 지적한 로그인 모달 문구만 우선 고친다.
  - B. 랜딩 전체의 상담 워크스페이스 문구를 모두 제거한다.
- 추천
  - A. 이번 문제는 로그인 모달의 잘못된 지원 대상 암시이므로 최소 범위로 수정한다.
