# 카드 복습 모드 정답보기 게이트 정비 (2026-06-08)

## 1차

### 작업내용

- 복습 모드 진입 직후에는 문제와 `정답보기` 버튼만 보이도록 정답/난이도 액션 노출 조건을 고정한다.
- `정답보기` 버튼 클릭 후 문제, 정답, 어려움/좋음/쉬움 액션이 함께 보이도록 유지한다.
- 복습 카드 우측 상단의 `자기평가형 복습` 문구를 제거하고 해당 위치를 `스킵` 버튼만 담당하게 한다.
- 웹 변경 시 모바일 패리티를 함께 확인하고 필요한 경우 공용 복습 패널 문구/구조를 맞춘다.

### 논의 필요

- 스킵 버튼은 정답 공개 전에도 현재 카드 건너뛰기가 가능해야 하는가?

### 선택지

1. 스킵 버튼을 항상 우측 상단에 노출한다.
2. 정답 공개 후에만 스킵 버튼을 노출한다.

### 추천

- 사용자가 우측 상단 스킵 버튼을 명시했으므로 1안을 따른다. 복습 저장 없이 넘기는 행동은 정답 공개 전에도 필요하다.

### 사용자 방향

- 우측 상단 자기평가형 복습 문구를 제거하고 스킵 버튼을 배치한다.
- 초기에는 문제와 정답보기 버튼만 보이고, 정답보기 후 문제/정답/난이도 버튼이 보이게 한다.
- PR 없이 즉시 main에 반영한다.
