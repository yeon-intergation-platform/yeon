# 게임 댓글 액션 BFF 경계 테스트 보강

## 1차

### 작업내용

- 게임 댓글 좋아요, 비밀댓글 확인, 댓글 삭제 route의 인증/입력 검증/오류 매핑 테스트를 추가한다.
- 기능 동작은 변경하지 않고 현재 BFF 계약을 테스트로 고정한다.

### 논의 필요

- 없음. 사용자 조작 API의 장애 경계를 줄이는 테스트 보강이다.

### 선택지

- A. 댓글 액션 route 단위 테스트를 추가한다.
- B. Spring client 테스트까지 확장한다.
- C. 브라우저 E2E로 댓글 UI까지 검증한다.

### 추천

- A. route 경계의 회귀를 빠르게 막을 수 있고 기능 변경이 없다.

### 사용자 방향

- 추천 기준으로 진행한다.
