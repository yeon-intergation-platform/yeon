# 카드 에디터 작성 코드블록 언어 선택 UI (2026-06-06)

## 배경
- 현재 카드 리치 에디터는 코드블록의 `language` attr 저장, 미리보기 드롭다운, 하이라이트를 지원한다.
- 하지만 작성 패널 내부 코드블록은 `코드 블록` 라벨만 보여 유저가 노션처럼 현재 코드블록의 언어를 직접 설정할 수 없다.

## 차수
### 차수 1
- 작업내용
  - Tiptap `codeBlock` 노드에 작성용 NodeView를 붙여 코드블록 헤더에 언어 드롭다운을 표시한다.
  - 드롭다운 변경 시 `codeBlock.attrs.language`를 갱신하고, 기존 마크다운 직렬화/미리보기 하이라이트 경로가 그대로 사용되게 한다.
  - 작성 패널의 고정 `pre::before` 라벨과 새 헤더 UI가 충돌하지 않도록 스타일을 정리한다.
- 논의 필요
  - 없음. 사용자는 노션처럼 작성 코드블록에서 언어를 설정하는 동작을 명시적으로 요청했다.
- 선택지
  - A. 미리보기 드롭다운만 유지하고 안내 문구 추가
  - B. 작성 에디터 코드블록 NodeView에 언어 드롭다운 추가
  - C. 상단 툴바에 현재 코드블록 언어 드롭다운 추가
- 추천
  - B. 사용자가 보는 코드블록 바로 안에서 언어를 고르는 노션형 UX와 가장 일치한다.
- 사용자 방향
  - "그렇게 해봐" 요청에 따라 B로 진행한다.
