# 슬라임 전투 검증 화면 위치/몬스터 표시 보정 백로그 (2026-05-16)

## 차수 1

- 작업내용: 3페이지 전투 검증 화면에서 초록 슬라임 스프라이트가 원본 체크보드/가이드 배경과 함께 잘못 잘려 보이는 문제를 수정한다.
- 논의 필요: 없음
- 선택지:
  - 기존 원본 이미지를 그대로 쓰고 CSS로 크기만 조정한다.
  - 검증 화면용 투명 배경 스프라이트 시트를 생성하고 정확한 7x2 프레임 기준으로 렌더링한다.
- 추천: 투명 배경 스프라이트 시트를 생성하고 실제 프레임 수를 source of truth로 반영한다.
- 사용자 방향: 초록 슬라임이 제대로 보여야 한다.

## 차수 2

- 작업내용: 핑크 슬라임 body, 초록 슬라임 hurtbox, HP bar, 스프라이트 렌더 기준을 바닥선 기준으로 재정렬한다.
- 논의 필요: 없음
- 선택지:
  - 디버그 박스를 기존처럼 반투명 면으로 덮는다.
  - 디버그 박스는 outline 중심으로 바꾸고 스프라이트 가시성을 우선한다.
- 추천: outline 중심 디버그 박스로 전환한다.
- 사용자 방향: 핑크/초록 슬라임 위치가 이상하지 않아야 한다.

## 차수 3

- 작업내용: lint/typecheck/domain test와 로컬 화면 캡처로 3페이지 표시 개선을 확인한다.
- 논의 필요: 없음
- 선택지:
  - 정적 검증만 수행한다.
  - 정적 검증 + Playwright 화면 캡처를 함께 수행한다.
- 추천: 정적 검증 + 화면 캡처를 함께 수행한다.
- 사용자 방향: 실제 화면 기준으로 확인한다.
