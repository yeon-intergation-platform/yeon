# 슬라임 빠른 검 베기 이펙트 개선 백로그 (2026-05-16)

## 1차

### 작업내용

- `/slime-game` 공격이 칼을 그냥 내리는 느낌으로 보이는 문제를 개선한다.
- 기존 칼+이펙트 결합 SVG를 제거하고, 사용자가 제공한 PNG 칼 본체 에셋과 칼끝 궤적 이펙트 에셋을 분리해 사용한다.
- 칼은 사용자가 제공한 PNG 디자인을 그대로 사용한다.
- 이펙트는 칼끝이 북쪽에서 동쪽으로 빠르게 이동하며 남기는 호 형태로 다시 만든다.
- attack action의 체감 속도를 높여 빠르게 베는 느낌이 나게 한다.

### 논의 필요

- 현재 차수는 시각 피드백 개선만 다룬다.
- 몬스터, 데미지, 히트박스, 피격 판정은 이번 범위에 포함하지 않는다.

### 선택지

- A. 검 본체 PNG와 칼끝 호 이펙트 PNG를 분리하고 stage에서 같은 attack progress를 기준으로 합성한다.
- B. 슬라임 sprite sheet 자체를 다시 그려 검을 포함시킨다.
- C. CSS border/gradient로 검과 호를 그린다.

### 추천

- A. 에셋 책임을 분리해 디자인 수정이 쉽고, 사용자가 지적한 “칼에 이펙트가 함께 붙은 문제”를 직접 해결한다.

### 사용자 방향

- A 기준으로 진행한다.
