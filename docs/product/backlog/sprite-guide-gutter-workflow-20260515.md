# 스프라이트 가이드 gutter 시트 워크플로우

## 1차

### 작업내용

- `/sprite-editor`에 AI 생성용 가이드 시트 규격을 명시한다.
- 64x64 프레임 16개 사이에 4px magenta(`#ff00ff`) gutter를 둔 1084x64 guide sheet를 생성/검수 입력 규격으로 삼는다.
- guide sheet import 시 gutter 영역을 버리고 16개 64x64 프레임만 추출한다.
- 최종 sprite sheet export는 gutter 없는 1024x64 투명 시트로 유지한다.

### 논의 필요

- 향후 8프레임/32프레임, 다른 프레임 크기, 2행 이상 시트까지 확장할지 결정 필요.

### 선택지

1. MVP는 16프레임, 64x64, 4px magenta gutter 고정 규격으로 구현한다.
2. 모든 규격을 사용자 입력으로 열어둔다.
3. 기존 불규칙 샘플 자동 분해를 계속 주력으로 개선한다.

### 추천

- 1번. 생성 규격과 절단 규격을 먼저 강하게 맞춰야 AI 스프라이트 QA 파이프라인이 안정된다.

### 사용자 방향

- 추천 기준으로 진행한다.
