# 오렌지 색상 전면 제거 백로그

## 1차: 지원 서비스와 공통 인증 UI의 오렌지 사용 제거

- 작업내용
  - 타자/카드/공통 헤더/공통 배지/인증 UI에서 오렌지 계열 hex와 Tailwind orange/amber/yellow 강조를 제거한다.
  - 디자인 시스템 문서에서 타이핑/인증 오렌지 허용 규칙을 삭제하고 검정/중립 토큰 중심으로 정리한다.
  - React Native/Universal UI 전환 계획에 맞게 웹 전용 오렌지 의미를 새로 만들지 않는다.
- 논의 필요
  - 경고/승리/팀 색상 같은 상태색은 향후 `@yeon/design-tokens` semantic token으로 별도 설계할지 여부.
- 선택지
  - A. 현재는 오렌지를 전부 검정/중립/보더 토큰으로 치환한다.
  - B. 일부 경고 상태만 amber 계열로 유지한다.
- 추천
  - A. 사용자가 오렌지 전면 금지를 명확히 지시했으므로 오렌지 계열을 남기지 않는다.
- 사용자 방향
  - 우리 서비스 어디에서도 주황색을 사용하지 않는다.
