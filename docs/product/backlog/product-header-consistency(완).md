# 제품 헤더 크기/설정 버튼 일관화 백로그

> 상태: 완료. 2026-05-12에 카드·타자·커뮤니티·홈 범위로 반영했다.

## 1차 — 플래시카드 덱 기준 헤더 프레임 통일

### 작업내용

- 타자연습, 플래시카드 덱, 커뮤니티, yeon.world 메인 화면의 상단 헤더 프레임 높이/패딩을 플래시카드 덱 홈 기준으로 맞춘다.
- 내부 링크/네비게이션/서비스별 문구는 각 서비스 특성을 유지한다.
- 공통 래퍼 컴포넌트로 크기 source of truth를 만든다.

### 논의 필요

- 헤더 내부 요소까지 완전히 동일하게 만들지는 않는다.
- 모바일에서 타자연습 네비게이션이 많아질 때 줄바꿈/가로 스크롤 유지 여부.

### 선택지

1. 각 화면 className을 수동으로 맞춘다.
2. 공통 `ProductHeader` 래퍼로 크기만 공유하고 내부 children은 서비스별로 유지한다.

### 추천

- 2번. 크기 source of truth를 한 곳으로 모으되 내부 요소는 통일하지 않는 사용자 방향과 맞다.

### 사용자 방향

- 플래시카드 덱 헤더 크기로 통일한다.
- 헤더 내부 요소는 통일하지 않는다.
- 설정 버튼 모양새는 typing-service 톱니 버튼 기준으로 통일한다.

## 2차 — 설정 버튼 외형 통일

### 작업내용

- typing-service의 톱니바퀴 설정 버튼 스타일을 공통 버튼 컴포넌트로 승격한다.
- card-service 설정 열기 버튼은 동일한 톱니바퀴 버튼 외형을 사용하고, 기존 설정 다이얼로그 동작은 유지한다.

### 논의 필요

- 버튼 라벨 노출 여부: 외형 기준은 톱니 아이콘 버튼으로 통일한다.

### 선택지

1. card-service 버튼 텍스트만 줄인다.
2. 공통 톱니 아이콘 버튼 컴포넌트를 사용한다.

### 추천

- 2번. 설정 버튼 모양 source of truth를 만든다.

### 사용자 방향

- typing-service에서 사용하는 설정 톱니바퀴 버튼대로 통일한다.
