# Todo 집중 화면 패널 정렬 보정

## 1차

### 작업내용

- Todo 집중 화면 타이머 패널의 폭, 내부 여백, 버튼 행 비율을 스크린샷 기준으로 정렬한다.
- 타이머 숫자, 프리셋 버튼, 직접 입력 행, 시작/리셋 버튼, 작업 완료 버튼의 기준선을 맞춘다.
- 기존 집중 화면 기능과 타임아웃 알림음 로직은 유지한다.

### 논의 필요

- 색상은 추가하지 않고 기존 집중 화면의 warm 스타일 값을 그대로 사용한다.
- 모바일에서는 버튼이 좁아지지 않도록 반응형 축소 규칙만 추가한다.

### 선택지

- A. CSS 정렬만 조정한다.
- B. 마크업 구조를 다시 나누고 컴포넌트를 분리한다.

### 추천

- A. 요청은 정렬 보정이므로 구조 변경 없이 CSS만 좁게 수정하는 것이 적절하다.

### 사용자 방향

- 추천 기준으로 진행한다.
