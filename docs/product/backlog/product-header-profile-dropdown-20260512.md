# 제품 헤더 프로필 드롭다운

## 1차

### 작업내용

- 공통 제품 헤더의 프로필 아이콘을 즉시 `/profile` 이동 링크에서 드롭다운 트리거로 바꾼다.
- 드롭다운 메뉴에 `내정보보기`, `로그아웃`을 제공한다.
- 로그아웃은 기존 `useLogout` 훅을 재사용해 세션 정리 source of truth를 유지한다.

### 논의 필요

- 별도 계정 정보 표시(이름/이메일)는 현재 요청 범위 밖이므로 추가하지 않는다.

### 선택지

1. 아이콘 클릭은 드롭다운, 내정보보기 항목만 `/profile`로 이동한다.
2. 아이콘 클릭도 이동하고 hover에서만 메뉴를 연다.
3. 기존 링크 유지 후 로그아웃 버튼을 다른 곳에 추가한다.

### 추천

- 1번. 사용자가 지적한 즉시 이동 문제를 직접 해결하고, 모바일/데스크톱 모두 클릭 동작으로 일관된다.
