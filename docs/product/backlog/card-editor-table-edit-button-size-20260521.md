# 카드 추가 모달 표 편집 버튼 크기 통일 (2026-05-21)

## 1차

### 작업내용

- 카드 추가 모달의 리치 에디터에서 표 선택 시 노출되는 표 편집 액션 버튼을 현재 `표 삽입` 툴바 버튼과 같은 정사각형 크기 기준으로 통일한다.
- 행 추가, 열 추가, 표 정렬 액션은 텍스트형 버튼 대신 아이콘형 버튼으로 유지하고, `aria-label`/`title`로 의미를 보존한다.

### 논의 필요

- 텍스트 라벨을 화면에 계속 노출할지, 툴바와 같은 아이콘 버튼으로 맞출지.

### 선택지

1. 높이만 맞추고 텍스트 버튼 폭은 유지한다.
2. `표 삽입` 버튼과 같은 정사각형 아이콘 버튼으로 통일한다.

### 추천

- 요청이 “같은 크기”이므로 2안으로 진행한다. 접근성 라벨과 hover title로 의미를 보완한다.

### 사용자 방향

- 현재 표 삽입 버튼과 같은 크기로 전부 통일한다.
