# support 홈 주요 문제 진입 개선

## 1차

### 작업내용

- 공개 콘텐츠 500단계 계획의 8차 support 디자인 시스템 중 176번을 진행한다.
- `support.yeon.world` 홈에 실제 공개 글로 연결되는 주요 문제 진입 영역을 추가한다.
- 없는 문서가 노출되지 않도록 실제 support article slug 기반 helper와 테스트를 둔다.

### 논의 필요

- 정적 글이 더 늘어나면 주요 문제 진입 순서를 검색 유입/클릭 데이터 기준으로 바꿀지 결정해야 한다.

### 선택지

- A. 실제 글 slug 기반의 주요 문제 바로가기만 먼저 추가한다.
- B. 검색 입력 UI를 먼저 만들고 실제 검색 기능은 나중에 연결한다.
- C. 홈 구조를 크게 바꿔 help center 전체 레이아웃을 다시 만든다.

### 추천

- A. 현재 단계에서는 동작하지 않는 검색 UI보다 실제로 이동 가능한 문제 진입이 사용자와 SEO 모두에 안전하다.

### 사용자 방향

- 비어 있으면 추천 기준으로 진행한다.
