# 공개 콘텐츠 품질 기준과 템플릿

작성일: 2026-06-17  
대상: `support.yeon.world`, `news.yeon.world`, `blog.yeon.world`  
범위: 운영 문서와 작성 템플릿. 상담 워크스페이스는 제외한다.

## 차수

### 24차: 콘텐츠 품질 기준과 작성 템플릿 고정

작업내용:

1. 공개 콘텐츠 품질 체크리스트를 공식 SEO 문서로 만든다.
2. 검색형 제목 작성 원칙을 체크리스트에 포함한다.
3. support 문서 템플릿을 만든다.
4. news 공지/업데이트 템플릿을 만든다.
5. blog 기술/제품 글 템플릿을 만든다.
6. 세 템플릿 모두 channel 역할과 금지 범위를 명시한다.

논의 필요:

- 템플릿을 실제 원고 import 포맷으로 바로 쓸지, 작성 가이드로만 둘지.

선택지:

- 작성 가이드, import 포맷, 둘 다.

추천:

- 지금은 작성 가이드로 두고, import workflow가 생기면 frontmatter 포맷을 별도로 확정한다.

사용자 방향:

- 비어 있으면 추천 기준으로 진행한다.
