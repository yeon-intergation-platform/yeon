# 슬라임 스프라이트 공중 프레임 추가 상승 백로그 (2026-05-16)

## 1차

### 작업내용

- `/sprite-editor` 슬라임 샘플의 4~5번 점프 프레임을 현재보다 몇 픽셀 더 위로 이동한다.
- guide sheet와 stripped sheet를 동일 기준으로 보정한다.

### 논의 필요

- 없음. 사용자 피드백에 따라 기존 공중 프레임의 점프감을 조금 더 강화한다.

### 선택지

- A. 1px만 추가 상승한다.
- B. 3px 추가 상승한다.
- C. 6px 이상 크게 추가 상승한다.

### 추천

- B. “몇 픽셀 정도” 요청에 맞춰 과하지 않게 점프감을 분명히 만든다.

### 사용자 방향

- "몇픽셀정도만 더 띄워지게 해줘."
