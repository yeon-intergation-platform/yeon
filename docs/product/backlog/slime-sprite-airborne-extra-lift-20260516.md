# 슬라임 스프라이트 8프레임 재제작 백로그 (2026-05-16)

## 1차

### 작업내용

- `/sprite-editor` 슬라임 샘플을 8프레임 기준으로 다시 제작한다.
- 3번/5번 프레임이 과하게 길쭉해 보이지 않게 stretch를 줄이고, 4~5번은 공중 프레임으로 분리한다.
- guide sheet와 stripped sheet를 동일한 프레임 기준으로 교체한다.

### 논의 필요

- 없음. 기존 샘플의 3번/5번 실루엣이 지나치게 길어 보인다는 사용자 피드백을 반영한다.

### 선택지

- A. 기존 프레임에서 3번/5번만 축소한다.
- B. 8프레임 전체를 같은 팔레트/윤곽/장식 기준으로 다시 제작한다.

### 추천

- B. 일부 프레임만 보정하면 실루엣과 장식 위치가 다시 흔들릴 수 있어, 8프레임 전체를 같은 규칙으로 재생성하는 편이 낫다.

### 사용자 방향

- "3번 5번이 너무 길쭉해지는거 같아. 다시 8장 뽑아줄래?"
