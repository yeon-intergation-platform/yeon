# 카드 편집기 선택 영역 코드버튼 범위 수정 백로그 (2026-06-06)

## 1차

### 작업내용

- 카드 리치 마크다운 편집기에서 일부 텍스트만 드래그한 뒤 코드 버튼을 눌렀을 때 선택한 부분만 코드로 바뀌도록 수정한다.
- 한 칸/일부만 선택했는데 위아래 문단까지 코드블록으로 바뀌는 범위 오염을 제거한다.
- 기존 코드블록 미리보기 언어 선택/하이라이트 동작은 유지한다.

### 논의 필요

- 없음. 선택 영역 기반 편집 명령은 선택 범위가 source of truth여야 한다.

### 선택지

1. 현재 selection range를 정확히 split해서 선택 노드만 코드블록으로 감싼다.
2. 블록 단위 전체 변환 정책을 유지하되 사용자에게 선택 범위를 문단 단위로 안내한다.

### 추천

- 1번. 드래그한 부분만 바뀌는 것이 사용자의 명시 요구와 일반 편집기 기대 동작에 맞다.

### 사용자 방향

- 추천 기준으로 진행한다.
