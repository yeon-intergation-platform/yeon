# 커뮤니티 X 스타일 타임라인 디자인

- 작성일: 2026-05-12
- 상태: 진행
- 범위: `apps/web/src/features/community/community-page.tsx`

## 1차

### 작업내용
- 커뮤니티 피드의 둥근 카드 반복 구조를 X/Twitter식 타임라인 구조로 바꾼다.
- 각 글은 아바타, 작성자명, 핸들, 시간, 카테고리, 본문, 액션바가 한 흐름으로 보이게 한다.
- 댓글도 카드 박스가 아니라 들여쓰기된 타임라인 답글처럼 보이게 한다.
- 우측 레일은 작성자 정보/인기 글/최근 댓글을 유지하되 피드와 대비되는 보조 정보 패널로 정리한다.

### 논의 필요
- 실제 이미지 첨부/미디어 카드 기능은 현재 API 계약에 없으므로 이번 차수에서는 시각 구조만 반영한다.

### 선택지
1. 현재 API 그대로 X식 텍스트 타임라인으로 개선한다.
2. 이미지/첨부/리트윗/좋아요 등 새 피드 기능까지 Spring/API에 추가한다.

### 추천
- 1번으로 즉시 디자인 밀도를 개선하고, 미디어/좋아요는 별도 백로그에서 Spring 소유 기능으로 추가한다.

### 사용자 방향
- 첨부 레퍼런스처럼 X/Twitter식 커뮤니티 피드로 바꾼다.
