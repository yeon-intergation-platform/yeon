# Today 중복 카드 제거 백로그

## 목표

`todo-service`에서 달력과 중복되는 날짜 이동 카드, Today 컬럼과 중복되는 "지금 할 일" 패널을 제거한다.

## 1차: 화면 카드 제거

### 작업내용

- 상단 우측의 날짜 이동 카드를 제거한다.
- "지금 할 일" 패널을 제거한다.
- 이미 `active` 상태인 일은 Today 컬럼에서 계속 보이게 유지한다.
- 달력의 날짜 선택과 월 이동은 유지한다.

### 논의 필요

- 날짜 하루 단위 이동을 완전히 없앨지, 달력 안에 보조 컨트롤로 남길지 여부.
- `active` 상태 전환을 계속 지원할지, 일반 Today 상태만 남길지 여부.

### 선택지

- A. 날짜 이동 카드와 지금 할 일 패널만 제거하고, 기존 상태 모델은 유지한다.
- B. `active` 상태 자체를 제거하고 모든 Today 일을 planned로 통합한다.
- C. 날짜 이동 컨트롤을 달력 내부에 축소 배치한다.

### 추천

A를 추천한다. 사용자가 지목한 중복 카드만 제거하면 UI가 단순해지고, 기존 localStorage 상태와 상태 전이 호환성도 유지된다.

### 사용자 방향

날짜 이동 카드는 달력이 있으니 없애고, "지금 할 일" 카드도 제거한다.

## 완료 조건

- 화면에서 날짜 이동 카드가 렌더링되지 않는다.
- 화면에서 "지금 할 일" 패널이 렌더링되지 않는다.
- 달력 날짜 선택과 Today/Inbox/Done 컬럼은 유지된다.
- active 상태의 일도 Today 컬럼에서 보인다.
