# 스프라이트 프레임 편집 툴 1차

## 1차

### 작업내용

- GPT 이미지 생성 결과나 업로드한 PNG를 브라우저에서 불러와 64x64 프레임 단위로 잘라 확인한다.
- 16프레임 걷기 시트를 기준으로 프레임별 확대 편집, 기준선/중심선 검수, 애니메이션 미리보기를 제공한다.
- 편집 결과를 PNG 스프라이트시트와 JSON 메타로 다운로드할 수 있게 한다.

### 논의 필요

- 이 툴을 장기 운영 관리자 기능으로 둘지, 프로토타입 제작용 내부 도구로 둘지 결정 필요.
- 향후 Aseprite 원본 파일까지 관리할지 여부는 별도 결정 필요.

### 선택지

1. 내부 독립 페이지로 먼저 제공한다.
2. 기존 타자 캐릭터 관리자에 통합한다.
3. 별도 데스크톱/CLI 제작 파이프라인으로 분리한다.

### 추천

- 1번. 지금은 이미지 생성 결과를 즉시 검수/보정하는 목적이므로 DB/API 없이 브라우저 로컬 도구로 시작한다.
