# 커뮤니티 실시간 채팅 닫힘 모션 백로그

## 1차수

### 작업내용

- compact 실시간 채팅 위젯을 닫을 때 본문이 남아 있는 상태에서 컨테이너 폭이 즉시 줄어드는 원인을 제거한다.
- 닫힘 동작은 본문 fade/blur exit 완료 후 컨테이너가 원형 버튼 크기로 줄어들게 순서를 분리한다.
- 열림 동작과 기존 UI/버튼 의미는 유지한다.

### 논의 필요

- 없음.

### 선택지

- A. 닫힘 duration만 늘린다.
- B. 본문 표시 상태와 컨테이너 접힘 상태를 분리해 exit 완료 뒤 폭을 줄인다.

### 추천

- B. 단순 duration 조정은 본문이 눌리는 원인을 남기므로 상태 전이를 분리한다.

### 사용자 방향

- 실시간 채팅방 닫힘이 갑자기 절반 폭으로 줄어들지 않고 자연스럽게 사라지게 한다.
