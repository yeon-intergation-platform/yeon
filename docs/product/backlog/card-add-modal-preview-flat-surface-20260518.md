# 카드 추가 모달 미리보기 단일 표면화 (2026-05-18)

## 배경

카드 추가 모달의 질문/답변 미리보기 영역이 외부 패널 안에 내부 카드 박스를 다시 두는 구조라, 카드 작성 흐름에서 불필요하게 중첩된 네모 박스로 보인다. 사용자는 미리보기 패널 자체가 하나의 카드 표면처럼 보이고 내부에는 추가 테두리 박스를 만들지 않는 구조를 원한다.

## 1차

### 작업내용

- 카드 추가 모달의 질문 미리보기에서 외부 미리보기 패널 안쪽 중첩 테두리/그림자 박스를 제거한다.
- 카드 답변 미리보기에도 동일한 단일 표면 구조를 적용한다.
- 기존 카드서비스 흰 배경/검정 텍스트/회색 보더 디자인 톤은 유지한다.
- 저장, 업로드, 마크다운 렌더링 동작은 변경하지 않는다.

### 논의 필요

- 미리보기 패널 상단 라벨과 본문 사이의 구분선은 유지할지 여부.
- 빈 상태 안내 문구를 패널 본문에 바로 표시할 때 시각적 위계가 충분한지 확인 필요.

### 선택지

1. 현재 컴포넌트 구조를 유지하고 내부 카드 박스 스타일만 제거한다.
2. 미리보기 컴포넌트를 별도 단일 표면 컴포넌트로 분리한다.
3. 질문/답변 미리보기를 완전히 다른 카드 레이아웃으로 재설계한다.

### 추천

1번. 요청 범위는 중첩 박스 제거이므로 구조 변경 없이 스타일만 좁게 수정해 회귀 위험을 줄인다.

### 사용자 방향

추천 기준으로 진행한다.
