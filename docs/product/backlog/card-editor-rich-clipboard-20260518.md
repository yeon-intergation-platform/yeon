# 카드 에디터 복합 클립보드/이미지 안정화 2026-05-18

## 1차

### 작업내용

- `study-platform-client` 공용 TipTap 에디터의 복합 클립보드/이미지 처리 방식을 Yeon 카드 에디터에 맞게 선별 이식한다.
- Yeon 카드 서비스의 기존 흰 배경 에디터, 좌우 작성/미리보기 레이아웃, 툴바 디자인, 업로드 API(`uploadCardDeckImage`)는 유지한다.
- 이미지 단독 붙여넣기뿐 아니라 텍스트+이미지 rich HTML 붙여넣기, data URI 이미지, 이미지 URL 붙여넣기, Notion attachment placeholder, HEIC/HEIF 보정/변환을 지원한다.

### 논의 필요

- 이번 차수는 표/YouTube/코드블록 고도화까지 확장하지 않고 이미지/클립보드 안정화에 집중한다.
- 카드 한 면당 이미지 최대 20개 정책은 유지한다.

### 선택지

1. 공용 에디터를 새로 만들고 카드 에디터를 교체한다.
2. 기존 카드 에디터 내부 유틸만 보강한다.
3. `study-platform-client` 코드를 거의 그대로 복사한다.

### 추천

- 2번. 디자인과 카드 서비스 UX를 유지하면서 기능 위험만 좁게 개선한다.

### 사용자 방향

- Yeon의 디자인은 완전히 유지하고, 가져올 만한 기능만 가져온다.
