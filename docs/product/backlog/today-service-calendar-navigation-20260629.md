# Today 서비스 날짜 이동과 달력 전체보기

## 목표

`todo.yeon.world`에서 오늘만 보는 MVP를 확장해 날짜별 이동과 월간 달력 전체보기를 제공한다. 사용자는 과거/미래 날짜를 선택해 그 날짜의 할 일을 추가·확인·완료할 수 있어야 한다.

## 1차

### 작업내용

- `selectedDate`를 실제 오늘 날짜와 분리한다.
- 이전/다음 날짜 이동, 오늘로 이동, 날짜 직접 선택을 제공한다.
- 할 일 추가, 진행중 지정, 완료, 복원은 선택한 날짜 기준으로 처리한다.
- 월간 달력에서 날짜별 계획/완료/남은 일 수를 요약하고 클릭 시 해당 날짜로 이동한다.
- 달력 월 이전/다음 이동을 제공한다.

### 논의 필요

- 장기 저장을 서버 DB로 옮길지 여부.
- 반복 일정, 시간대, 알림 같은 캘린더 고급 기능을 넣을지 여부.

### 선택지

- A. 기존 localStorage 모델을 확장해 날짜별 보기와 월간 요약을 먼저 제공한다.
- B. Spring API/DB 스키마를 먼저 만든 뒤 서버 기반 달력으로 확장한다.
- C. 외부 캘린더 연동까지 포함해 설계를 크게 잡는다.

### 추천

선택지 A를 추천한다. 사용자가 바로 쓰는 개인용 서비스가 목표이고, 현재 MVP의 저장 모델이 이미 `plannedFor`와 `completedOn`을 가지고 있어 날짜 이동과 월간 요약을 작은 변경으로 만들 수 있다.

### 사용자 방향

날짜별 이동과 달력 전체보기가 모두 되게 만든다.

## 완료 조건

- 날짜 이동 버튼과 날짜 선택 입력으로 다른 날짜를 볼 수 있다.
- 월간 달력에서 날짜별 상태 요약을 보고 날짜를 선택할 수 있다.
- 선택 날짜 기준으로 Today/Done/진행중/완료율이 갱신된다.
- 기존 localStorage 데이터와 호환된다.
- 테스트와 로컬 화면 검증을 통과한다.
