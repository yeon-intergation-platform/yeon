# 스프라이트 가이드 샘플 캐릭터 적용 백로그 (2026-05-15)

## 1차

### 작업내용

- 64x64 16프레임 + 4px magenta gutter 규격에 맞는 샘플 캐릭터 guide sheet를 추가한다.
- `/sprite-editor`의 기본 샘플 로딩을 실제 guide sheet 절단 흐름으로 바꿔 사용자가 바로 재생 검수할 수 있게 한다.

### 논의 필요

- 없음. 사용자가 현재 템플릿 기준으로 캐릭터를 뽑고 편집기에 적용하라고 요청했다.

### 선택지

- A. 기존 불규칙 샘플 시트를 유지하고 자르기만 보정한다.
- B. guide gutter 규격에 맞는 새 샘플 시트를 추가하고 기본 샘플을 교체한다.

### 추천

- B. 생성 규격과 절단 규격 일치를 검증하는 목적에 맞다.

### 사용자 방향

- "만들어 그럼", "이거로 캐릭터 뽑아줘. 그리고 sprite-editor에 적용해줘 확인할 수 잇도록"
