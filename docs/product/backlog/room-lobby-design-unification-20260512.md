# 방 로비 디자인 통일 백로그 (2026-05-12)

## 1차: 카드방 기준으로 타자방 로비 상단 구조 통일

### 작업내용

- 카드방/타자방 로비가 같은 "방" 경험으로 보이도록 상단 구조를 맞춘다.
- 타자방 로비에 카드방과 같은 우측 입장 캐릭터 카드와 필터/검색/방 만들기 CTA 배치를 적용한다.
- 카드방 프로필 카드 문구도 스크린샷 기준의 `입장 캐릭터` / `캐릭터 바꾸기`로 정리한다.

### 논의 필요

- 없음. 사용자 지적은 두 방 로비의 통일감 부족이며, 기능 변경 없이 레이아웃과 문구만 맞추면 된다.

### 선택지

1. 타자방을 카드방 레이아웃에 맞춘다.
2. 카드방을 타자방 레이아웃에 맞춘다.
3. 별도 공통 컴포넌트를 즉시 추출한다.

### 추천

- 1번. 스크린샷의 카드방 구조가 사용자가 제시한 기준이며, 타자방 기능을 건드리지 않고 같은 방 로비 패턴으로 맞출 수 있다.

### 사용자 방향

- 비어 있음. 추천 기준으로 진행한다.
