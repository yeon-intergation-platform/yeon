# 카드 덱 상세 화면 폭 축소 백로그 (2026-06-08)

## 1차

### 작업내용

- 카드 덱 상세 화면의 본문/헤더 최대 가로폭을 현재 1280px 기준의 약 0.8배인 1024px로 축소한다.
- 카드 목록 영역이 너무 넓어 보이는 문제를 줄이고, 기존 흰 배경/검정 CTA/회색 보더 디자인 규칙은 유지한다.

### 논의 필요

- 모바일 화면까지 동일하게 줄일지 여부.

### 선택지

1. 웹 상세 화면 컨테이너 최대폭만 1024px로 줄인다.
2. 카드 행 내부 여백/폰트/삭제 컬럼까지 함께 줄인다.
3. 모바일 상세 화면도 별도로 폭/여백을 줄인다.

### 추천

- 1번. 사용자가 제시한 HTML은 웹 카드 덱 상세 화면이고, 레이아웃 chrome은 패리티 레지스트리상 platform-divergent이므로 웹 컨테이너 최대폭만 1024px로 줄이는 것이 가장 작고 안전하다.

### 사용자 방향

- 현재보다 0.8배 정도의 가로 크기로 축소한다.
