# 공개 콘텐츠 관리자 최근 큐 보강

작성일: 2026-06-17  
대상: `yeon.world/admin/content`  
범위: 공개 콘텐츠 운영 대시보드 읽기 전용 보강. 상담 워크스페이스는 제외한다.

## 차수

### 22차: 최근 발행, 최근 수정, SEO 경고 큐

작업내용:

1. Spring admin read API에서 받은 글 행을 기준으로 최근 발행 글 목록을 계산한다.
2. Spring admin read API에서 받은 글 행을 기준으로 최근 수정 글 목록을 계산한다.
3. SEO 경고가 있는 글만 별도 운영 큐로 분리한다.
4. `/admin/content` 대시보드에 세 큐를 읽기 전용으로 노출한다.
5. `/admin/content/[channel]` 화면에도 채널별 SEO 경고 큐를 노출한다.
6. 생성, 수정, 삭제, 발행, archive 버튼은 계속 제공하지 않는다.

논의 필요:

- 최근 큐 노출 개수는 몇 개로 제한할지.

선택지:

- 5개, 10개, 전체.

추천:

- 초기 운영 화면에서는 5개만 보여주고 전체 확인은 기존 글 목록에서 처리한다.

사용자 방향:

- 비어 있으면 추천 기준으로 진행한다.
