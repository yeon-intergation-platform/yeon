## 기능 범위 정리

- 화면 ID: `TYPING_ROOM_LOBBY`
- 핵심: 설정 확인/수정, 참여자 상태 확인, 준비, 채팅, 초대, 시작
- 주요 규칙
  - 방장: 설정 수정 가능, 자동 준비완료
  - 참가자: 설정 보기 + 준비/준비취소
  - 시작 조건 충족 후 방장 시작 가능
  - 진행 중인 방 입장 제한
  - 긴 문구(MVP 규칙/시작 안내) 상시 노출 제거
  - PC 3단, 모바일 수직 스택
