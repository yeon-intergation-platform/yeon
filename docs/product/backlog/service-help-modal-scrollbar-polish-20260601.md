# 서비스 도움말 모달 스크롤바 정리 백로그

## 1차: 도움말 모달 내부 스크롤바를 단정하게 정리

- 작업내용
  - 서비스 도움말 모달의 기본 브라우저 스크롤바가 바깥 선처럼 보이지 않도록 내부 스크롤 영역과 여백을 분리한다.
  - 흰 배경 서비스 톤에 맞는 얇고 둥근 스크롤바 스타일을 적용한다.
  - 모달 모서리와 스크롤 트랙이 시각적으로 충돌하지 않게 한다.
- 논의 필요
  - 공통 스크롤바 클래스로 승격할지 여부.
- 선택지
  - A. 도움말 모달 전용 클래스만 추가한다.
  - B. 기존 counseling 중심 `.scrollbar-subtle`을 전 서비스용으로 바꾼다.
- 추천
  - A. 기존 상담 워크스페이스 스크롤바에 영향 주지 않고, 지적된 도움말 모달만 최소 수정한다.
- 사용자 방향
  - 도움말 모달만 최소 범위로 정리한다.
