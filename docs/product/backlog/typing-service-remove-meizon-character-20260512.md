# 타자서비스 메이존 캐릭터 제거

## 1차

### 작업내용

- 타자서비스 캐릭터 레지스트리 원천에서 `meizon` / `메이존` 캐릭터를 제거한다.
- 빌드 시 재생성되는 캐릭터 레지스트리에도 메이존이 노출되지 않도록 한다.
- 레이스 seed/protocol 로직은 변경하지 않는다.

### 논의 필요

- 기존 사용자가 이미 `meizon`을 선택한 상태일 때 fallback 캐릭터로 내려갈 수 있으나, 별도 마이그레이션 정책은 이번 요청 범위에 포함하지 않는다.

### 선택지

1. 원천 JSON만 삭제하고 레지스트리는 빌드 스크립트로 재생성한다.
2. UI에서만 숨기고 원천 데이터는 유지한다.
3. API/스토리지 마이그레이션까지 추가한다.

### 추천

- 1번. 캐릭터 제거 요구에는 원천 제거가 맞고, UI 숨김은 추후 다시 노출될 수 있는 거짓 상태를 만든다.

### 사용자 방향

- “타자서비스에 메이존 캐릭터 없애줘.”
