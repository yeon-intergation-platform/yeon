# 로그인 모달 색상 보정 및 로컬 디버깅 지침 반영 백로그

## 1차: 로그인 모달 색상을 서비스 톤에 맞추고 프로젝트 내부 지침으로 고정

- 작업내용
  - 로그인 모달의 따뜻한 크림/다크 그레이 계열 색을 랜딩·카드·타자 서비스의 흰 배경 톤에 맞게 정리한다.
  - 로컬 디버깅 세션 운영 규칙은 사라질 수 있는 외부 임시 메모리가 아니라 프로젝트 내부 AI 상시지침에 반영한다.
  - 잘못 생성된 외부 임시 메모리 노트를 제거한다.
- 논의 필요
  - 소셜 로그인 버튼의 브랜드 색상까지 서비스 톤으로 바꿀지 여부.
- 선택지
  - A. 모달 컨테이너/보조 UI만 흰 배경 톤으로 정리하고 소셜 버튼 브랜드 색은 유지한다.
  - B. 소셜 버튼까지 모두 흑백 CTA로 통일한다.
- 추천
  - A. 카카오/구글 버튼은 사용자 인지와 OAuth 관례가 중요하므로 유지하고, 모달 표면 색만 서비스 톤으로 맞춘다.
