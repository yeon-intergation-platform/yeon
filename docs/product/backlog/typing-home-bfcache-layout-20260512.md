# 타자연습 홈 뒤로가기 표시 안정화 백로그 (2026-05-12)

## 1차 - 프로필 패널 기본 렌더링과 세로 패딩 축소

- 작업내용: `/typing-service`에 브라우저 뒤로가기로 복귀해도 `내 프로필`/`오늘의 시작` 패널이 사라지지 않도록 localStorage 로딩 완료 여부로 전체 패널 렌더링을 막지 않는다. main 세로 패딩은 40px에서 20px로 줄인다.
- 논의 필요: localStorage 읽기 전에는 기본 프로필이 잠깐 보일 수 있다.
- 선택지: 1) 로딩 완료 전 패널 숨김 유지 2) 기본 프로필로 즉시 렌더링 후 저장 프로필로 갱신 3) 별도 스켈레톤 추가
- 추천: 2) 뒤로가기 복귀에서 핵심 CTA가 사라지는 것보다 기본 프로필 즉시 렌더링이 안전하다.
- 사용자 방향: 뒤로가기로 와도 프로필과 오늘의 시작이 보여야 하며, 세로 패딩은 20px로 줄인다.
