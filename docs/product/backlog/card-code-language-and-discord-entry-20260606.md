# 카드 코드 언어 선택 및 Discord AI 진입 카드 — 2026-06-06

## 1차

### 작업내용

- `https://yeon.world` 메인 서비스 목록에 `https://discord-ai.yeon.world`로 이동하는 Discord AI 카드 추가.
- 카드 저장/추가 모달에서 마크다운 코드블록 입력 중 미리보기 재렌더링으로 인한 흔들림 완화.
- 카드 미리보기 코드블록 헤더의 `code` 표시를 언어 선택 드롭다운으로 바꾸고, 선택 언어에 맞춰 코드블록 마크다운/미리보기 표시를 갱신.
- 질문/답변 편집 패널에서도 코드블록 영역이 일반 텍스트와 구분되도록 시각 표시 추가.

### 논의 필요

- 코드 하이라이트 라이브러리 신규 의존성 추가 여부.
- 언어 선택 정보의 저장 위치: 기존 마크다운 fence 언어(````js`)를 SSOT로 유지할지 별도 메타데이터를 추가할지.

### 선택지

1. 기존 마크다운 fence 언어를 SSOT로 유지하고, Tiptap codeBlock의 `language` 속성만 갱신한다.
2. 카드 데이터 모델에 코드블록 언어 메타데이터를 별도로 추가한다.
3. 외부 하이라이트 라이브러리를 새로 추가한다.

### 추천

- 1번. 카드 저장 계약을 바꾸지 않고, 기존 마크다운 fence 언어를 그대로 SSOT로 사용한다. 신규 의존성은 피하고 현재 렌더러 안에서 보수적인 토큰 하이라이트를 제공한다.
