# Todo 집중 화면 중앙 타이머 정리

## 1차

### 작업내용

- Todo 집중 화면에서 히어로 문구, 마음저널 배지, 장식 카드 이미지를 제거한다.
- 작업 타이머 패널만 화면 중앙에 배치한다.
- 집중 시간 최소값을 5분에서 1분으로 낮추고 1분 단위로 조정 가능하게 한다.
- 타이머가 0초가 되면 짧은 타임아웃 알림음을 재생한다.

### 논의 필요

- 알림음은 외부 파일을 추가하지 않고 브라우저 Web Audio API로 짧게 생성한다.
- 사용자가 명시한 화면 단순화 범위 안에서만 수정하고 Todo 보드 기능은 건드리지 않는다.

### 선택지

- A. 기존 배경 영상은 유지하고 중앙 패널만 남긴다.
- B. 배경 영상까지 제거하고 완전한 흰 배경 도구 화면으로 바꾼다.

### 추천

- A. 사용자가 좋아한 창문/바람 느낌은 유지하고, 방해되는 카드와 문구만 제거하는 것이 목적에 맞다.

### 사용자 방향

- 추천 기준으로 진행한다.
