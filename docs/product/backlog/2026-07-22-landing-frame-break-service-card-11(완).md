# 랜딩 타자연습 카드 제목 전경 간격 보정

> 상태: 완료

## 1차 — 서비스별 전경 세로 위치 적용

- 작업내용: 프레임 브레이크 설정에 전경 하단 위치 클래스를 두고 타자연습 카드만 아래로 이동한다.
- 논의 필요: 다른 카드 배치를 유지하면서 전등 상단과 제목 사이의 여백을 확보해야 한다.
- 선택지:
  - A. 카드별 전경 위치를 설정하고 타자연습만 24px 아래로 이동한다.
  - B. 모든 카드의 헤더 높이를 늘린다.
- 추천: A. 백지학습·플래시카드 덱의 검증된 구도를 바꾸지 않는다.
- 사용자 방향: 제목과 이미지 사이에 여백을 두어 전등 잘림을 해결한다.

## 2차 — 시각 및 코드 검증

- 작업내용: 사용자 화면과 가까운 넓은 카드 및 375px 모바일에서 전등 전체 노출을 확인한다.
- 논의 필요: 전경 하단이 과도하게 가려지거나 CTA 영역을 침범하지 않아야 한다.
- 선택지:
  - A. `-bottom-8`로 제한된 간격만 추가한다.
  - B. 임의 px 값으로 크게 이동한다.
- 추천: A. Tailwind 기본 scale을 사용해 작은 변경으로 해결한다.
- 사용자 방향: CSS로 가짜 전등을 만들지 않는다.

## 완료 조건

- 전등 상단과 제목 사이에 시각적 여백이 생긴다.
- 실제 전경 이미지 전체를 아래로 이동할 뿐 전등 도형을 생성하지 않는다.
- 다른 두 카드의 전경 위치는 유지된다.
- 모바일·데스크톱에서 가로 스크롤이 없다.

## 완료 결과

- 프레임 브레이크 설정에 `foregroundBottomClassName`을 추가해 전경 위치를 서비스별로 관리한다.
- 타자연습 카드만 `-bottom-8`을 사용해 기존보다 24px 아래로 이동했다.
- 백지학습·플래시카드 덱은 기존 `-bottom-2` 위치를 유지했다.
- 전등 에셋과 카드 구조는 변경하지 않았으며 전등 생성·연장 CSS도 추가하지 않았다.
- 375px와 1040px에서 제목과 전등 사이 여백, 전등 전체 노출, 가로 스크롤 부재를 확인했다.
