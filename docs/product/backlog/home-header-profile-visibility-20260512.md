# 홈 헤더/내정보/숨김 서비스 정리 백로그

## 1차수

### 작업내용

- 홈 공통 헤더에서 서비스 바로가기 버튼과 타자/플래시카드/커뮤니티 직접 링크를 제거한다.
- 상담 기록 워크스페이스는 홈 서비스 목록에서 기본 숨김 상태로 두고, 사용자가 토글로 다시 볼 수 있게 한다.
- 내정보 아이콘이 상담 서비스로 이동하지 않고 `/profile` 내정보 페이지로 이동하게 수정한다.
- 서비스 화면별 헤더 브랜드를 `YEON 타자연습`, `YEON 카드`, `YEON 커뮤니티`로 노출한다.

### 논의 필요

- `/profile`에서 추후 커뮤니티 프로필, 카드/타자 기록까지 통합할지 여부는 별도 확장 시 결정한다.

### 선택지

1. 내정보 아이콘을 기존 상담 서비스 진입으로 유지한다.
2. 내정보 아이콘을 신규 `/profile` 페이지로 연결하고 현재 계정 정보를 보여준다.

### 추천

- 2안. 아이콘 의미와 실제 이동 목적지를 일치시켜 사용자 혼선을 제거한다.

### 사용자 방향

- 내정보 아이콘은 내 정보를 보여줘야 하며, 상담 서비스 이동은 버그다.
- 숨김 버튼 문구는 `숨김 서비스 보기`로 변경한다.
- 서비스별 헤더 브랜드 라벨을 명확히 표시한다.
