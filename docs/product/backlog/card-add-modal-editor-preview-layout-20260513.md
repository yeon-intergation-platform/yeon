# 카드 추가 모달 에디터/미리보기 레이아웃 개선 (2026-05-13)

## 배경

카드 추가 모달의 질문/답변 리치 에디터에서 툴바가 2줄로 내려갈 때 작성 영역이 답답해 보이고, 오른쪽 미리보기 카드가 작은 박스처럼 보여 작성 결과 확인 영역으로 충분히 작동하지 않는다.

## 1차

### 작업내용

- 카드 추가 모달의 직접 작성 화면 가로 폭을 넓힌다.
- 에디터 툴바가 2줄이 되어도 자연스럽도록 툴바 패딩, 버튼 크기, 줄 간격을 정리한다.
- 질문/답변 에디터 본문 높이를 늘리고 답변 영역은 질문보다 더 크게 둔다.
- 오른쪽 미리보기 박스와 내부 카드의 최소 높이를 늘리고, 좌우 비율을 작성 55~60% / 미리보기 40~45%에 가깝게 조정한다.
- 질문/답변 반복 섹션의 정렬과 여백을 유지한다.

### 논의 필요

- 모달 폭 확대가 작은 화면에서 과도하게 보이지 않는지 확인이 필요하다.
- 미리보기와 에디터 높이를 동일하게 강제할지, 답변만 더 크게 둘지 판단이 필요하다.

### 선택지

1. CSS 클래스 중심으로 현재 컴포넌트 구조를 유지하며 높이/비율만 조정한다.
2. 에디터/미리보기 레이아웃 컴포넌트를 별도로 분리해 구조까지 리팩터링한다.
3. 미리보기를 별도 고정 패널로 분리한다.

### 추천

1번. 현재 문제는 구조 결함보다 공간 배분 문제이므로, 컴포넌트 API에 작은 size variant만 추가하고 기존 저장/업로드 동작은 건드리지 않는다.
