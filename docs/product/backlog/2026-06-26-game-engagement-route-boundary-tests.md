# 게임 참여 BFF 경계 테스트 보강

## 1차

### 작업내용

- 게임 허브의 좋아요, 찜, 댓글 BFF route에 인증/입력 검증/Spring 오류 매핑 테스트를 추가한다.
- 기능 동작은 변경하지 않고 현재 route 계약을 테스트로 고정한다.

### 논의 필요

- 없음. 운영 누락을 줄이는 테스트 보강이며 사용자 기능 변경이 아니다.

### 선택지

- A. 좋아요/찜/댓글 route 경계 테스트를 추가한다.
- B. Spring client 단위 테스트까지 넓힌다.
- C. 화면 E2E까지 확장한다.

### 추천

- A. route 경계가 운영 장애로 드러나는 지점이므로 1시간 범위에서 가장 효율적이다.

### 사용자 방향

- 추천 기준으로 진행한다.
