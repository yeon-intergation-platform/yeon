# 카드 에디터 코드블록 언어/복사 UX 적용 백로그 (2026-05-18)

## 배경

`study-platform-client` 마크다운 편집기 분석에서 가져올 가치가 높은 다음 후보로 코드블록의 언어 표시와 복사 UX를 선정했다. Yeon 카드 에디터의 기존 흰 배경·미니멀 디자인은 유지하고, 카드 작성/미리보기에서 코드 예시를 더 안정적으로 다룬다.

## 1차

### 작업내용

- 카드 에디터 툴바에 코드블록 토글을 추가한다.
- Markdown fenced code 붙여넣기를 TipTap codeBlock 노드로 정규화한다.
- 카드 프리뷰의 Markdown/HTML code block에 언어 라벨을 표시한다.
- 카드 프리뷰의 code block에 복사 버튼을 제공한다.
- HTML sanitizer는 code block language class만 안전하게 유지한다.

### 논의 필요

- 언어 선택 드롭다운까지 즉시 추가할지 여부.
- syntax highlighting 라이브러리를 도입할지 여부.

### 선택지

1. 툴바 토글 + 붙여넣기 정규화 + 라벨/복사만 구현한다.
2. 언어 선택 드롭다운과 하이라이터까지 한 번에 구현한다.
3. 표시 UX만 개선하고 에디터 입력 경험은 그대로 둔다.

### 추천

1번. 새 의존성 없이 기존 디자인을 유지하면서 실사용 효용이 높은 범위를 먼저 배포한다.
