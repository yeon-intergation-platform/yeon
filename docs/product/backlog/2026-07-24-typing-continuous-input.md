# 타자 연속 입력과 백스페이스 복구

## 차수 1

- 작업내용
  - 타자 연습과 멀티플레이 입력에서 정확히 입력한 접두부를 삭제 불가로 만드는 제약을 제거한다.
  - 빠른 연속 입력과 한글 IME 조합 결과를 첫 불일치 뒤에서 잘라내지 않고 문장 길이까지만 수용한다.
  - 백스페이스 키 반복으로 입력 전체를 자연스럽게 지울 수 있게 한다.
  - 입력 정규화 유틸에 연속 입력, 오타 포함 입력, 반복 삭제, 최대 길이 테스트를 추가한다.
- 논의 필요
  - 기존의 완료 접두부 고정 정책을 유지할지 여부.
- 선택지
  - (A) 기존 고정 정책을 유지하고 키 이벤트를 별도로 보정한다.
  - (B) 입력은 브라우저 기본 동작을 따르고 문장 최대 길이만 제한한다.
- 추천
  - (B). 네이티브 textarea 동작을 보존해 IME와 키 반복을 방해하지 않고, 정확도와 완료 여부는 기존 파생 계산으로 판정한다.
- 사용자 방향
  - 추천 기준으로 진행.
