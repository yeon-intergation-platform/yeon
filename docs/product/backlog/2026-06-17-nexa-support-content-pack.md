# NEXA support 콘텐츠 확장 백로그

작성일: 2026-06-17
대상: `support.yeon.world`

## 1차

### 작업내용

- NEXA 사용자가 바로 검색할 가능성이 높은 FAQ/가이드/정책형 support 글을 추가한다.
- 상담관리 서비스는 유지보수 대상이 아니므로 포함하지 않는다.
- 글 내용은 `discord-assitant`의 실제 README, i18n, 안전 정책, 유저 플로우 문서에서 확인 가능한 사실만 사용한다.
- sitemap/admin 통계는 공개 콘텐츠 registry에서 파생되게 유지한다.

### 논의 필요

- 지금 단계에서 support 글은 정적 registry로 관리한다.
- 관리자 화면에서 글 본문 수정/삭제 기능은 아직 만들지 않는다.
- 무료/유료 범위는 실제 문구가 바뀔 수 있으므로 확정 판매 문구처럼 과장하지 않는다.

### 선택지

- 선택지 A: NEXA support 글 6개를 먼저 추가한다.
- 선택지 B: NEXA, 타자, 카드, 커뮤니티를 각각 조금씩 추가한다.
- 선택지 C: admin CMS 기능을 먼저 만든다.

### 추천

- 선택지 A를 추천한다. 사용자가 직접 언급한 초기 support 글 목록에 NEXA 항목이 많고, `discord-assitant` 소스 문서가 현재 가장 구체적이다.

### 사용자 방향

- news, blog, support를 분리한다.
- support는 서비스별 FAQ/가이드/문제 해결 구조로 운영한다.
- 상담관리는 제외하고 현재 운영 대상 서비스와 NEXA를 다룬다.
