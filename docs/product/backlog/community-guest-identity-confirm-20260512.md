# 커뮤니티 비회원 작성자 확인 모달 백로그

## 1차수

### 작업내용

- 비회원 글 작성, 글 수정, 글 삭제, 댓글 작성, 댓글 삭제를 실행할 때 닉네임/비밀번호 확인 모달을 한 번 표시한다.
- 모달에서 `다시 보지 않음`을 선택하고 해당 요청이 성공하면 이후 같은 브라우저에서는 모달 없이 기존 작성자 정보 입력값으로 요청한다.
- 모달에서 입력한 닉네임/비밀번호는 전역 작성자 정보에 반영해 작성 요청과 서버 검증 요청이 같은 값을 쓰게 한다.

### 논의 필요

- 없음.

### 선택지

- A. 각 삭제/수정 버튼 옆에 매번 입력칸을 둔다.
- B. 첫 보호 작업에서만 모달을 띄우고, 다시 보지 않음으로 숨긴다.

### 추천

- B. 사용자 흐름을 방해하지 않으면서 비회원 수정/삭제 검증 의미를 명확히 전달한다.

### 사용자 방향

- 모달은 한 번만 뜨게 하고, 다시 보지 않음을 선택하면 모달이 뜨지 않게 한다.
