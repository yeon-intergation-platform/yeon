# 스프라이트 샘플 프레임 component 기반 분해 수정

## 1차

### 작업내용

- `/sprite-editor` 샘플 시트 분해가 전체 캐릭터 영역 16등분으로 동작해 프레임별 포즈가 잘리거나 인접 프레임이 섞이는 문제를 수정한다.
- 체크무늬 배경을 제외한 실제 캐릭터 픽셀의 column component를 찾아 16개 프레임별 crop 영역을 만든다.
- 각 프레임을 독립 crop 후 기준 캔버스에 맞춰 넣어 thumbnail/preview에서 몸이 잘리지 않게 한다.

### 논의 필요

- 실제 AI 업로드 프레임에는 시트 자동 분해가 아니라 개별 파일 업로드를 기본으로 둘지, 향후 시트 import UI를 별도 기능으로 확장할지 결정 필요.

### 선택지

1. 샘플 시트에 한정해 component 기반 자동 분해를 적용한다.
2. 모든 업로드 이미지에 시트 자동 분해를 적용한다.
3. 수동 crop 설정 UI를 추가한다.

### 추천

- 1번. 지금 문제는 샘플 검수용 시트 분해 버그이므로 MVP 범위를 넘기지 않고 샘플 로더만 정확하게 고친다.

### 사용자 방향

- 추천 기준으로 진행한다.
