# 타자 점령전 카드 소유권 보드 보정 백로그

## 1차: 중앙 카드 보드가 소유권을 직접 표현하도록 교체

- 작업내용
  - 점령전 플레이 화면의 중앙 엔진 영역을 카드 그리드로 교체한다.
  - 화면 배경을 빨강/파랑으로 반분하지 않고, 각 카드의 owner가 빨강/파랑/중립 색을 갖도록 표시한다.
  - 카드 그리드를 5x5로 빽빽하게 배치하고 단어가 카드 중앙에 보이도록 한다.
- 논의 필요
  - 카드 뒤집기 애니메이션을 CSS로 먼저 둘지, 엔진/캔버스 레이어에서 구현할지 여부.
- 선택지
  - A. React 카드 그리드로 소유권을 명확히 표시한다.
  - B. 기존 캔버스 엔진 배경을 유지하고 색만 조정한다.
- 추천
  - A. 사용자가 지적한 문제는 배경색이 아니라 카드 소유권 표현이므로, 중앙 카드 보드를 화면 source of truth인 snapshot에서 직접 렌더링한다.
- 사용자 방향

## 2차: 상대팀 카드 입력 규칙을 타깃 안내에 반영

- 작업내용
  - 내 팀이 빨강이면 파랑 카드가 우선 타깃으로 안내되도록 타깃 선택 순서를 조정한다.
  - 상대팀 카드가 없을 때만 중립 카드를 안내하고, 내 팀 카드 입력은 막는다.
- 논의 필요
  - 첫 시작부터 양 팀 카드가 배분되어야 하는지, 중립 카드에서 시작해야 하는지 여부.
- 선택지
  - A. 현재 서버 규칙은 유지하고, 상대팀 카드 우선 타깃 안내만 반영한다.
  - B. 서버 보드 초기 상태를 양 팀 카드로 미리 배분한다.
- 추천
  - A. 현재 프로토콜을 깨지 않고 즉시 사용자 이해와 조작 흐름을 맞춘다.
- 사용자 방향
