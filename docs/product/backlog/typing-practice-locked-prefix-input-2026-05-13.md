# 타자연습 입력 UX: 완료 구간 고정 + 복붙/커트 제어

## 차수 1

- 작업내용
  - `타자연습/타자방` 입력 로직에서 "이미 정확히 입력 완료된 앞부분"을 조작 불가 구간으로 고정한다.
  - 사용자가 완료 구간 이전을 지우거나 수정하지 못하도록 하여, 완료 텍스트는 고정된 상태로 다음 구간만 이어서 입력되도록 한다.
  - 완료 구간을 기준으로 렌더링 표시를 강화해 시각적으로 완료 구간이 안정적으로 보이도록 정리한다.
  - paste/copy/cut 이벤트를 차단해 복사·붙여넣기·잘라내기로 인한 입력 흐름 깨짐을 최소화한다.

- 논의 필요
  - 단일 화면(연습/멀티) 공통 처리 범위를 현재 최소 범위(연습 + 멀티 텍스트 입력 박스)로 할지 여부

- 선택지
  - (A) 연습 화면만 우선 적용
  - (B) 연습 화면과 멀티플레이어 입력 박스 동시 적용

- 추천
  - 기존 구조를 크게 바꾸지 않기 위해 (B)로 공통 유틸을 두어 동작 일관성 확보

- 사용자 방향
  - (B) 선택 반영
