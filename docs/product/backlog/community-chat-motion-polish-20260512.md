# 커뮤니티 채팅 위젯 모션 개선 백로그

## 1차수

### 작업내용

- compact 채팅 위젯의 접힘/펼침 전환에서 컨테이너 크기, 아이콘, 본문 exit 모션을 함께 제어한다.
- 닫힐 때 본문이 갑자기 사라지거나 부모 레이아웃이 먼저 튀는 느낌을 줄인다.
- 모션 민감 사용자를 위해 reduced motion 분기를 둔다.

### 논의 필요

- 없음.

### 선택지

- A. 본문 `height/opacity`만 유지한다.
- B. 컨테이너 `layout/width/height`와 아이콘 전환까지 함께 제어한다.

### 추천

- B. 실제로 사용자가 보는 접힘 과정은 컨테이너와 아이콘까지 포함되므로 함께 제어한다.

### 사용자 방향

- framer-motion을 대충 쓰지 말고 사라지는 과정까지 개선한다.
