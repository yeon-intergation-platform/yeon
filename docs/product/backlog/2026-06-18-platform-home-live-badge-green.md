# 플랫폼 홈 운영 상태 뱃지 색상 개선

## 1차

### 작업내용

- 플랫폼 홈 서비스 카드의 `운영 중` 상태 뱃지를 회색이 아닌 초록색 뱃지로 보이게 한다.
- `준비 중` 상태는 기존 회색 계열을 유지한다.
- 흰 배경 디자인 규칙을 유지하면서 새 임의 hex 색상은 추가하지 않는다.

### 논의 필요

- 초록 상태색을 전체 서비스 상태 컴포넌트의 공용 토큰으로 승격할지는 별도 디자인 시스템 정비 때 판단한다.

### 선택지

- A. 현재 뱃지 분기에서 `live` 상태에만 Tailwind `green-*` 상태색을 적용한다.
- B. 새 hex 색상을 직접 추가한다.
- C. 공용 상태 뱃지 컴포넌트를 새로 만든다.

### 추천

- A. 요구가 작고 명확하며, B는 색상 규칙에 어긋나고 C는 현재 범위보다 과하다.

### 사용자 방향

- 추천 기준으로 진행한다.

### 실행 결과

- 플랫폼 홈 `운영 중` 상태 뱃지를 초록 배경/테두리/텍스트/점으로 변경한다.
- 기존 `YeonBadge` neutral 스타일 충돌을 피하기 위해 해당 상태 뱃지만 `span` 렌더링으로 직접 제어한다.
