# 슬라임 스프라이트 공중 프레임 보정 백로그 (2026-05-16)

## 1차

### 작업내용

- `/sprite-editor` 슬라임 샘플의 4~5번 프레임이 점프 중인데도 바닥선에 붙어 보이는 문제를 수정한다.
- 실제 재생용 stripped sheet와 guide sheet의 4~5번 프레임을 같은 기준으로 위로 이동해 공중 프레임으로 보이게 한다.

### 논의 필요

- 없음. 점프/바운스 중간 프레임은 바닥선에서 분리되어야 한다.

### 선택지

- A. UI에서 4~5번 프레임만 렌더링 오프셋을 둔다.
- B. 샘플 에셋 자체의 4~5번 프레임 위치를 보정한다.

### 추천

- B. 프레임별 기준선 검수 대상은 에셋 자체이므로, 잘못된 샘플을 UI에서 감추면 실제 export 품질을 오해하게 된다.

### 사용자 방향

- "슬라임이 4번 5번에서 점프를 안해. 왜지? 점프하면 공중에 띄워져있어야하는데 하단에 붙어있음."
