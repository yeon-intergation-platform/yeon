# Todo 할 일 카드 뱃지 드롭다운 편집

## 1차

### 작업내용

- Todo 할 일 카드의 우선순위 뱃지를 드롭다운으로 바꿔 `높음`, `보통`, `낮음`을 바로 수정할 수 있게 한다.
- 예상 시간 뱃지를 드롭다운으로 바꿔 `15분`, `30분`, `1시간`, `2시간+`을 바로 수정할 수 있게 한다.
- 변경값은 해당 할 일의 원본 `priority`, `estimate`에 저장하고 이득 점수와 집중 화면 진입 시간에도 반영되게 한다.

### 논의 필요

- 추천 카드의 뱃지는 목록 미리보기 성격이므로 이번에는 직접 편집 대상으로 보지 않는다.
- 화면 색상은 기존 Todo 흰 배경 디자인 시스템 값만 사용한다.

### 선택지

- A. native `select`를 뱃지 형태로 스타일링해 접근성과 구현 안정성을 우선한다.
- B. 커스텀 팝오버 드롭다운을 새로 만든다.

### 추천

- A. 요구는 간단한 설정 변경이므로 native select가 KISS 원칙에 맞다.

### 사용자 방향

- 추천 기준으로 진행한다.
