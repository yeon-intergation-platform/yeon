# 타이핑 서비스 메뉴 카드 hover UI 개선

## 1차

### 작업내용

- 타이핑 서비스 홈의 메뉴 카드에서 기본 상태에는 메뉴명만 노출한다.
- 부가 설명은 브라우저 `title` tooltip이 아니라 카드 내부 커스텀 hover UI로 렌더링한다.
- hover/focus 시 카드 하단 설명이 부드럽게 fade-in 되며 카드 높이와 주변 레이아웃은 흔들리지 않게 한다.

### 논의 필요

- 모바일 터치 환경에서 hover가 없으므로 설명은 focus 상태에서도 접근 가능하게 할지 여부.

### 선택지

1. hover 전용으로만 설명 노출
2. hover + keyboard focus 상태에서 설명 노출

### 추천

- 2번. 기본 화면은 메뉴명만 유지하면서 키보드 접근성도 확보한다.

### 사용자 방향

- 추천 기준으로 진행한다.
