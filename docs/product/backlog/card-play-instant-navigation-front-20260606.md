# 카드 플레이 이동 시 즉시 앞면 표시 백로그 (2026-06-06)

## 1차수

### 작업내용
- 카드가 뒷면인 상태에서 이전/다음 문제로 이동해도 뒤집힘 애니메이션 없이 새 카드 앞면이 즉시 보이게 한다.
- 클릭/Space/Enter로 같은 카드를 뒤집을 때만 기존 flip transition을 유지한다.

### 논의 필요
- 없음. 사용자가 원하는 이동 UX가 명확하다.

### 선택지
1. 인덱스 변경 순간에 flip transition을 비활성화하고 앞면 상태를 렌더링한다.
2. 이전/다음 버튼 클릭 전에 먼저 앞면으로 되돌리는 애니메이션을 실행한 뒤 이동한다.

### 추천
- 1번. 사용자가 싫어한 “갑자기 뒤집히는 느낌”을 제거하고, 문제 이동은 즉시 앞면 표시라는 기대와 맞는다.

### 사용자 방향
- 즉시 작업 및 커밋 요청. 추천 기준으로 진행한다.
