# 카드 에디터 Mermaid 렌더링 적용 백로그 (2026-05-18)

## 배경

표, YouTube, 코드블록 복사 UX 다음 단계로 학습 카드 안에서 흐름도/관계도/시퀀스 다이어그램을 표현할 수 있게 한다. Yeon의 기존 카드 에디터 디자인은 유지하고, Mermaid는 프리뷰 표현력 확장 범위로 제한한다.

## 1차

### 작업내용

- `mermaid` 의존성을 웹 앱에 추가한다.
- Markdown fenced code의 `mermaid` 언어 블록을 카드 프리뷰에서 다이어그램으로 렌더링한다.
- HTML 저장값의 `<pre><code class="language-mermaid">`도 동일하게 렌더링한다.
- 렌더 실패 시 기존 코드블록/오류 안내로 안전하게 후퇴한다.
- 새 에디터 UI 톤은 만들지 않고 기존 흰 배경·얇은 테두리·작은 버튼 스타일을 유지한다.

### 논의 필요

- Mermaid 편집 보조 UI를 별도 버튼/템플릿으로 제공할지 여부.
- 복잡한 다이어그램 확대/전체화면 UX를 추가할지 여부.

### 선택지

1. 프리뷰 렌더링만 먼저 적용한다.
2. Mermaid 삽입 버튼과 템플릿까지 함께 적용한다.
3. Mermaid는 보류하고 표 편집 UX를 먼저 강화한다.

### 추천

1번. 새 의존성이 들어가므로 프리뷰 렌더링과 안전한 실패 경계만 먼저 배포한다.
