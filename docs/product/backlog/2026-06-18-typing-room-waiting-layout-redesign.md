# 타자방 대기 화면 레이아웃 정리

## 1차

### 작업내용

- 타자방 점령전 대기 화면을 상단 방 요약, 좌측 방 정보, 중앙 팀 대기실, 하단 채팅 구조로 재정렬한다.
- 기존 방 상태, 참가자, 준비, 점령전 입장 로직은 유지하고 TSX 마크업과 Tailwind 배치만 조정한다.
- 큰 화면에서 UI가 과도하게 퍼지지 않도록 최대 폭, 고정 슬롯 높이, 반응형 그리드를 정리한다.

### 논의 필요

- 팀 이동 버튼은 현재 별도 팀 교체 API가 없으므로 이번 차수에서는 시각적 위치만 유지한다.
- 기존 설정 패널/참가자 패널/음성 패널은 화면 하단 보조 영역으로 남길지, 추후 점령전 대기 전용 화면에서 숨길지 별도 판단이 필요하다.

### 선택지

- 선택지 A: 점령전 대기 전용 패널만 재배치한다.
- 선택지 B: 대기 화면 전체 구조를 새 컴포넌트로 분리하고 기존 보조 패널까지 재설계한다.

### 추천

- 선택지 A를 먼저 적용한다. 현재 요청은 시각 배치 문제 해결이 핵심이고, 로직 분리나 API 변경은 불필요하다.

### 사용자 방향

- 사용자가 제공한 시안처럼 상단 요약, 좌측 방 정보, 중앙 팀 대기실, 하단 채팅이 한 화면에서 자연스럽게 보이도록 정리한다.
