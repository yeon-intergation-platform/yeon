# 랜딩 Today 전등 헤더 겹침 재보정

> 상태: 완료

## 1차 — 전등 상단 안전 영역 확대

- 작업내용: YEON Today 전경을 추가로 내려 전등 소켓 상단이 번호 레이어 뒤에 들어가지 않게 한다.
- 논의 필요: 기존 16px 보정만으로는 전등 소켓 상단이 번호 하단과 여전히 겹친다.
- 선택지:
  - A. 번호 레이어의 z-index를 낮춘다.
  - B. 5번 전경만 16px 더 내려 실제 여백을 확보한다.
- 추천: B. 제목 가독성과 다른 카드 레이어를 유지하면서 문제만 해결한다.
- 사용자 방향: 전등이 어느 부분도 번호 때문에 잘려 보이지 않게 한다.

## 2차 — 실제 화면 확인

- 작업내용: localhost의 모바일·데스크톱 화면에서 전등 소켓 상단과 번호 사이의 간격을 확인한다.
- 논의 필요: 없음.
- 선택지:
  - A. 코드 수치만 확인한다.
  - B. Playwright 스크린샷으로 실제 합성 결과를 확인한다.
- 추천: B.
- 사용자 방향: localhost에서 바로 확인한다.

## 완료 조건

- 5번 번호와 전등 소켓 상단 사이에 명확한 흰 여백이 보인다.
- 인물과 할 일 보드 구도는 유지된다.
- 이미지 에셋과 다른 카드 위치는 변경하지 않는다.

## 완료 결과

- 5번 전경을 추가로 16px 내려 전등 소켓 상단과 번호 사이에 흰 안전 여백을 확보했다.
- 인물·할 일 보드의 카메라 구도와 다른 카드 위치는 유지했다.
