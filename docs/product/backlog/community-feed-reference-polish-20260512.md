# 커뮤니티 피드 레퍼런스형 타임라인 다듬기

## 1차

### 작업내용

- 커뮤니티 피드를 첨부 레퍼런스처럼 카드 묶음보다 얇은 구분선 기반 타임라인으로 더 강하게 정리한다.
- 글 본문은 제목 카드 느낌을 줄이고, 작성자 행 아래 본문 텍스트가 바로 이어지는 구조로 조정한다.
- 액션바 간격, 아바타/핸들/시간 위계, 우측 보조 영역을 레퍼런스형 피드에 맞게 낮은 대비로 다듬는다.

### 논의 필요

- 현재 API에는 이미지 첨부, 번역 배지, 실제 리포스트/좋아요/조회수 원천이 없으므로 이번 차수에서는 시각 구조와 기존 피드 동작을 우선한다.

### 선택지

1. 현재 데이터 구조 안에서 레퍼런스형 타임라인 밀도를 높인다.
2. 이미지/좋아요/리포스트 API까지 추가해 완전한 소셜 피드로 확장한다.
3. 기존 카드형 UI를 유지한다.

### 추천

- 1번. 사용자가 지적한 “천편일률적인 커뮤니티” 문제는 우선 피드 밀도와 위계 조정으로 바로 개선할 수 있다.

### 사용자 방향

- X/Twitter 레퍼런스처럼 보이는 커뮤니티 디자인으로 수정.
