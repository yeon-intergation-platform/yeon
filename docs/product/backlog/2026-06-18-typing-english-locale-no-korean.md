# 타자 서비스 영어 locale 한글 노출 제거

## 배경

영어 언어 설정(`locale=en`)에서도 타자 서비스 주요 화면에 한국어 UI 문구가 남아 있다. 사용자는 영어 설정 시 한국어가 한 글자도 보이지 않는 동작을 요구했다.

## 1차

### 작업내용

- `typing-service`의 locale 상태와 번역 소스 구조를 확인한다.
- 홈, 자유 연습, 레이스, 방 목록/생성/대기, 덱 목록/설정처럼 사용자가 바로 접근하는 주요 화면의 raw 한국어 UI 문구를 locale 기반 문구로 교체한다.
- Playwright로 영어 설정을 적용한 뒤 주요 타자 서비스 화면에 한글이 남는지 검사한다.

### 논의 필요

- 관리자 전용 캐릭터 프레임/덱 운영 화면까지 즉시 완전 영문화할지, 사용자가 보는 일반 경로부터 막을지 선택이 필요하다.

### 선택지

- A. 일반 사용자 경로부터 영어 UI를 완성하고 관리자 전용 화면은 후속 백로그로 분리한다.
- B. `typing-service` 아래 모든 파일의 모든 사용자 노출 문자열을 한 번에 영문화한다.

### 추천

A. 현재 요구의 핵심은 영어 설정 사용자가 실제 서비스에서 한글을 보지 않는 것이다. 일반 사용자 경로를 먼저 Playwright로 검증하고, 관리자 전용 운영 화면은 별도 차수로 다루는 편이 변경 범위와 검증 비용을 통제할 수 있다.

### 사용자 방향

추천 기준으로 진행한다.

## 완료 기준

- 영어 설정 후 주요 타자 서비스 화면에서 visible text 기준 한글 정규식(`[가-힣]`)이 검출되지 않는다.
- 관련 타입 검사와 lint가 통과한다.
