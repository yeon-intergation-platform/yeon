# 타자 기본 덱을 알쓸신잡 지식으로 전환

## 차수 1

- 작업내용
  - 한국어 로컬 기본 덱을 타자 훈련 문구에서 알아두면 쓸모 있는 지식 문장으로 교체한다.
  - QR 코드, 블루투스, 웹, 화성, 원자시계, 문어, GPS, 이모지처럼 일상과 기술에 가까운 소재를 섞는다.
  - 덱 제목과 설명, API 실패 시 fallback 문구까지 같은 지식 덱 정체성으로 통일한다.
  - 사실 문장은 공식 기관과 기술 원소유자 자료로 검증한다.
- 논의 필요
  - 영어 기본 덱도 같은 방향으로 바꿀지 여부.
- 선택지
  - (A) 한국어와 영어 기본 덱을 모두 교체한다.
  - (B) 사용자가 보고 지적한 한국어 기본 덱만 우선 교체한다.
- 추천
  - (B). 요청 범위에 맞춰 한국어만 바꾸고 영어 덱은 기존 학습 흐름을 유지한다.
- 사용자 방향
  - 한국어 기본 덱을 알쓸신잡형 지식으로 변경.
