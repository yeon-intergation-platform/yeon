# News 문서 템플릿

## 사용 범위

`news.yeon.world`의 공식 공지, 제품 업데이트, 업계 뉴스 해설에 사용한다. support 사용법이나 blog 회고를 news로 대신 발행하지 않는다.

## 공지/업데이트 구조

```md
# 사실 중심 제목

## 요약

무엇이 바뀌었는지 2-3문장으로 쓴다.

## 적용 대상

- 서비스:
- 적용일:
- 영향을 받는 사용자:

## 무엇이 바뀌었나요

변경 전과 변경 후를 구체적으로 적는다.

## 사용자에게 영향이 있나요

사용자가 직접 해야 할 일이 있는지 적는다.

## 필요한 조치

조치가 없으면 "별도 조치는 필요하지 않습니다"처럼 명확히 쓴다.

## 관련 링크

- 관련 support 문서
- 관련 blog 글
```

## 업계 해설 구조

```md
# Yeon 사용자에게 의미가 보이는 제목

## 이 소식이 왜 중요한가요

외부 소식을 짧게 요약한다.

## Yeon 또는 NEXA와 어떤 관련이 있나요

사용자, 운영자, 개발자 관점의 의미를 적는다.

## 당장 바뀌는 것은 무엇인가요

서비스 정책이나 기능 변경이 없으면 없다고 쓴다.

## 참고 링크

출처 링크를 짧게 남긴다.
```

## 발행 전 체크

- [ ] 공식 공지, 업데이트, 업계 해설 중 하나로 분류된다.
- [ ] 적용 서비스와 사용자 영향이 명확하다.
- [ ] support 문서를 반복하지 않고 링크로 연결한다.
- [ ] 외부 원문을 길게 복사하지 않았다.
