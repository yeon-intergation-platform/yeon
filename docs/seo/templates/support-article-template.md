# Support 문서 템플릿

## 사용 범위

`support.yeon.world`의 사용법, 튜토리얼, 문제 해결, FAQ 문서에 사용한다. 상담 워크스페이스 문서는 만들지 않는다.

## 작성 구조

```md
# 검색형 제목

## 언제 이 문서를 보나요

이 문서가 해결하는 상황을 한두 문장으로 적는다.

## 시작하기 전에 확인할 것

- 필요한 권한
- 로그인 또는 게스트 상태
- 사용할 서비스 URL

## 해결 순서

1. 첫 번째로 열 화면이나 설정을 적는다.
2. 사용자가 눌러야 할 버튼이나 확인할 값을 적는다.
3. 성공 여부를 확인하는 방법을 적는다.

## 실패하면 확인할 것

- 가장 흔한 실패 원인
- 권한 또는 네트워크 문제
- 다른 support 글로 이어지는 링크

## 다음에 할 일

- 제품 진입 CTA
- 관련 FAQ
- 관련 공지 또는 업데이트
```

## 제목 예시

- 디스코드 서버에 NEXA AI 봇 추가하는 방법
- NEXA 봇이 응답하지 않을 때 확인할 5가지
- card.yeon.world에서 플래시카드 덱을 만드는 방법

## 발행 전 체크

- [ ] 사용자가 그대로 따라 할 단계가 있다.
- [ ] 실패했을 때 확인할 항목이 있다.
- [ ] 관련 support 문서로 내부 링크가 연결된다.
- [ ] title과 description이 검색 문장에 가깝다.
