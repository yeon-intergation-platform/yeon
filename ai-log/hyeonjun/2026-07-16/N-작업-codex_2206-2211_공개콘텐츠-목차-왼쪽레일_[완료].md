# 공개 콘텐츠 목차 왼쪽 레일 작업 로그

## 작업 범위

- News·Support·Blog 상세 글의 데스크톱 본문 목차를 왼쪽 sticky 레일로 분리한다.
- 본문 폭과 시작선은 목차 유무에 영향받지 않도록 유지한다.
- 중간 폭·모바일은 접이식 목차를 유지한다.

## 제약

- 로컬 브랜치에서만 수정한다.
- PR, push, `main` 병합, 배포는 하지 않는다.
