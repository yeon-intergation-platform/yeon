# 블로그 인플레이스 필터 작업 로그

## 작업 범위

- 블로그 홈 카테고리 버튼을 컬렉션 페이지 링크에서 URL 기반 필터로 바꾼다.
- 선택해도 헤로·필터 위치·스크롤 위치를 유지하고 카드 목록만 바꾼다.
- URL 새로고침과 직접 접근에서도 선택 상태가 복구되게 한다.

## 제약

- 로컬 브랜치에서만 수정한다.
- PR, push, `main` 병합, 배포는 하지 않는다.
- UI 변경 후 Playwright로 전환·새로고침·스크롤 보존을 확인한다.

## 진행 상태

- [완료] 현재 필터가 `/blog/<category>` 컬렉션 페이지로 이동해 전체 화면이 교체되는 원인을 확인했다.
- [완료] `router.replace`와 `scroll: false`를 사용해 `/blog?category=<category>` URL 기반 인플레이스 필터를 구현했다.
- [완료] 선택된 카테고리는 해당 글 전체를 표시하고, `전체`는 기존 최근 글 목록을 유지한다.

## 결과

- `제품 글` 클릭은 블로그 홈에서 필터 버튼으로 동작한다.
- URL과 새로고침은 선택 상태를 유지하고, 이전처럼 `/blog/product` 컬렉션 화면으로 이동하지 않는다.
- 로컬 커밋만 진행하며 PR, push, `main` 병합, 배포는 하지 않는다.
