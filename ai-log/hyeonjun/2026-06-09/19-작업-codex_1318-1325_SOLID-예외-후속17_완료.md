# SOLID/예외 후속 17

## 목표

- 300개 SOLID/예외 백로그 항목 100~108을 완료한다.
- 웹 카드 서비스 인증/카드방/게스트 덱 count 실패 메시지를 구체화한다.

## 변경

- race-server 카드방 leave/request catch는 16차 helper 반영 상태를 확인하고 완료 처리했다.
- typing-race-room catch 후보는 현재 코드에서 제거된 상태를 확인했다.
- 웹 커뮤니티 post 404 catch는 처리 가능한 404만 null로 변환하고 나머지는 재전파하는 경계로 유지했다.
- `MissingCardServiceAuthProviderError`를 추가해 Provider 누락 원인과 조치 위치를 구체화했다.
- 웹 카드방 생성/입장 catch 메시지를 helper로 통합했다.
- 게스트 덱 개수 확인 실패를 helper로 변환하고 화면 실패 메시지로 연결했다.
- 백로그 항목 100~108을 완료 처리했다.
