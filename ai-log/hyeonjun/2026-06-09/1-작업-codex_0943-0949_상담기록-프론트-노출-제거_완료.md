# 상담기록 프론트 노출 제거

## 목표

- 구글 검색, 브라우저 탭, 메인 화면에서 사용자가 `상담기록` 용어를 보지 않도록 프론트 노출 문자열을 제거한다.

## 제약

- 상담 워크스페이스는 동결 서비스이므로 기능 로직 변경 없이 노출 문구 중심으로 최소 수정한다.
- 변경 후 main 배포 흐름까지 진행한다.

## 진행

- 09:43 작업 시작. `yeon-2`에서 `origin/main` 기반 브랜치 생성.

## 결과

- 루트 SEO title/description/keywords/OG/JSON-LD에서 `상담기록`/`상담 기록` 노출 제거.
- 메인 서비스 목록에서 상담 워크스페이스를 공개 목록에서 제외하고 숨김 토글도 제거.
- 상담 워크스페이스 내부 사용자 노출 문구와 AI 프롬프트의 `상담 기록` 표현을 `운영 메모` 계열로 교체.
