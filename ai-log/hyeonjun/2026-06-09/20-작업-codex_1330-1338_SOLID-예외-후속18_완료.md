# SOLID/예외 후속 18

## 목표

- 300개 SOLID/예외 백로그 항목 109~118을 완료한다.
- 웹 카드 에디터 이미지/클립보드/게스트 이관 실패 메시지를 구체화한다.

## 변경

- 이미지 정규화 실패 메시지에서 string/기타 unknown 원인을 보존한다.
- 코드 블록/마크다운 코드 복사 실패 메시지에 복사 대상 길이와 브라우저 권한 원인을 포함한다.
- 덱 내보내기 복사 실패를 throw 대신 사용자 오류 메시지 상태로 연결한다.
- 마크다운 콘텐츠 동적 복사 버튼 catch는 오류를 console.warn에 남긴다.
- 게스트 덱 이관 catch 메시지를 `getMergeGuestDialogErrorMessage`로 통합한다.
- 카드 에디터 이미지 업로드/교체/URL 붙여넣기 실패 메시지를 helper로 통합한다.
- 백로그 항목 109~118을 완료 처리했다.

- `CI=true pnpm --filter @yeon/web typecheck`
