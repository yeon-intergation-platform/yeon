## 구현

- CSS로 추가했던 검은 캡을 제거했다.
- built-in `imagegen`으로 기존 전등의 잘린 상단 관절과 지지대를 생성했다.
- 생성 이미지에서 전등 상단 영역만 추출·크로마키 제거·디스필한 뒤 기존 투명 전경에 합성했다.
- 인물·책상·전등 갓의 기존 픽셀은 유지하고 복원 범위를 전등 상단으로 제한했다.
- 복원된 전경 한 장이 카드 헤더와 이미지 영역을 함께 가로지르도록 중복 레이어를 통합했다.
- 제목 묶음에 흰색 안전 배경과 오른쪽 패딩을 적용했다.

## 생성 프롬프트 요약

- 기존 캐릭터·책상·전등 갓은 그대로 유지한다.
- 초록색 빈 영역에서 잘린 검은 전등 목과 대각 지지대를 자연스럽게 연장한다.
- 둥근 관절과 짧은 지지대로 캔버스 안에서 완결하고 CSS 도형처럼 보이는 캡은 만들지 않는다.
- 배경은 단색 `#00ff00`, 텍스트·UI·워터마크는 추가하지 않는다.
