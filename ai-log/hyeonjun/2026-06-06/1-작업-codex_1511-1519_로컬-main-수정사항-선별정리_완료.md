# 1. 로컬 main 수정사항 선별 정리 (완료)

## 작업시간

- 2026-06-06 15:11 ~ 15:19

## 목적

- 로컬 `main`에 남아 있던 미커밋 변경을 원격 `main` 기준으로 안전하게 정리한다.
- 대량 스킬 삭제처럼 위험한 변경은 제외하고, 보존 가치가 있는 문서 산출물만 별도 브랜치로 선별 반영한다.

## 차수

### 차수 1

- 작업내용
  - 로컬 미커밋 변경 전체를 stash 백업으로 보존.
  - `origin/main` 최신 커밋으로 로컬 `main`을 fast-forward.
  - 백엔드 유지보수 점검 문서와 해당 작성 로그만 새 브랜치에 복원.
- 논의 필요
  - 없음
- 선택지
  - A. 모든 로컬 변경을 그대로 PR에 포함
  - B. 문서/로그만 선별하고 스킬 삭제 및 이미 원격 반영된 AGENTS 변경 제외
- 추천
  - B. 원격 main 기준 충돌과 스킬 소실 위험을 줄이는 선별 반영
- 사용자 방향
  - "정리해줘" 요청에 따라 추천안으로 진행

## 제외한 변경

- `.codex/skills` 대량 삭제: 원격 main에 파일들이 존재하므로 안전하지 않아 제외.
- `AGENTS.md` race-server 검증 문구: 원격 main에 이미 반영되어 제외.
- `.agents/skills/*` description 변경: 현재 요청 범위와 직접 관련이 낮아 제외.
