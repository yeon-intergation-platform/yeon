# 프로필 등록 폼 간격과 닉네임 입력 디자인 개선

- 시작: 22:52
- 종료: 23:04
- 상태: 완료
- 대상: `/profile`의 `ProfileEditSection`
- 목표: 사진·닉네임·저장 흐름의 간격을 정돈하고 직사각형 닉네임 입력을 주변 UI와 일관된 형태로 바꾼다.
- 시각 검증: 데스크톱·모바일 before/after Playwright 스크린샷

## 변경 결과

- 넓은 화면에서는 사진 편집과 닉네임 편집을 두 열로 정렬하고, 작은 화면에서는 한 열로 쌓이도록 구성했다.
- 닉네임 라벨을 입력창 위의 블록 요소로 고정해 라벨과 입력창이 겹치지 않게 했다.
- 닉네임 입력을 높이 48px의 캡슐형 필드로 바꾸고 포커스 테두리와 링을 추가했다.
- 입력·저장 흐름을 같은 열에 묶어 시선 이동과 간격을 단순화했다.

## 시각 검증

- 데스크톱: 입력 크기 `420x48`, 라벨 간격 `8px`, 계산된 반경 `9999px`
- 모바일: 입력 크기 `250x48`, 라벨 간격 `8px`, 계산된 반경 `9999px`
