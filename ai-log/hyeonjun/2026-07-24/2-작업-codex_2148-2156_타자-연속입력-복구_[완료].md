# 타자 연속 입력 복구

## 차수 1

- 작업내용
  - 타자 입력이 빠른 연속 키 입력과 백스페이스 반복을 그대로 반영하도록 입력 경계를 수정한다.
  - 입력 정규화 회귀 테스트와 실제 브라우저 동작을 검증한다.
- 논의 필요
  - 없음.
- 선택지
  - 완료 접두부 고정 유지 / 브라우저 기본 편집 동작 복원.
- 추천
  - 브라우저 기본 편집 동작 복원.
- 사용자 방향
  - 추천 기준으로 진행.
- 실행 상태
  - 완료.

## 변경

- `typing-input-utils.ts`
  - 완료 접두부 삭제 금지와 첫 오타 뒤 입력 절단을 제거했다.
  - 브라우저가 전달한 최신 입력을 문장 길이까지만 정규화한다.
- solo/multiplayer 화면
  - 공통 입력 정규화 함수를 사용한다.
- 회귀 테스트
  - 빠른 연속 입력, 오타 뒤 입력, 반복 삭제, 최대 길이 경계를 검증한다.
