# 타자 알쓸신잡 기본 덱

## 차수 1

- 작업내용
  - 한국어 기본 타자 덱을 상식적이면서 신기하고 현대적인 지식 문장으로 교체한다.
  - 덱 메타데이터와 fallback 문구의 정체성을 함께 맞춘다.
- 논의 필요
  - 없음.
- 선택지
  - 훈련 문구 일부 교체 / 한국어 기본 덱 전체 전환.
- 추천
  - 한국어 기본 덱 전체 전환.
- 사용자 방향
  - 알쓸신잡형 지식 덱으로 변경.
- 실행 상태
  - 완료.

## 변경

- 한국어 로컬 기본 덱 제목을 `알쓸신잡 지식`으로 변경했다.
- 기존 타자 훈련 문구 3개를 기술·과학·문화·생물 분야 지식 문장 8개로 교체했다.
- API fallback 문장은 기본 덱 첫 문장을 같은 SSOT에서 참조하도록 통일했다.
- 영어 기본 덱과 사용자 덱은 변경하지 않았다.

## 사실 검증

- DENSO WAVE: QR 코드 오류 정정과 최대 30퍼센트 복구 단계.
- Bluetooth SIG: 하랄 블로탄 이름과 H·B 룬 문자 로고.
- CERN: 최초 웹사이트와 NeXT 컴퓨터.
- NASA: 화성 일몰의 푸른빛.
- NIST: 세슘 주기 기반 1초와 GPS 상대론.
- Smithsonian Ocean: 두족류의 세 심장과 구리 기반 푸른 피.
- Unicode Consortium: 일본 통신사 이모지의 유니코드 편입.
