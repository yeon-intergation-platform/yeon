# 투두 기록 우클릭 메뉴와 간결한 설명 편집

## 요청

- 기록 카드의 좌클릭 동작 충돌을 피하도록 OS식 우클릭 드롭다운을 제공한다.
- 큰 설명 편집 패널을 더 간단하게 만든다.

## 작업 범위

- `apps/web/src/features/today/today-record-screen.tsx`
- 관련 테스트
- 데스크톱·모바일 Playwright 시각 증거

## 설계 결정

- 좌클릭은 기록 추가라는 주 동작으로 유지한다.
- 우클릭과 더보기 버튼은 동일한 공용 `YeonContextMenu`를 연다.
- 편집 UI는 활동 선택·한 줄 설명·취소·저장만 남긴다.
- API 계약, 저장 규칙, 공통 배경이나 다른 서비스는 변경하지 않는다.

## 검증

- Playwright 실제 브라우저
  - 우클릭 메뉴 열림: 통과
  - 우클릭 시 기록 수 불변: 통과
  - 좌클릭 두 번째 기록 추가: 통과
  - 모바일 더보기 메뉴: 통과
  - 모바일 가로 오버플로: `0px`
- 정적·자동 검증
  - `pnpm --filter @yeon/web lint`: 통과
  - `pnpm --filter @yeon/web typecheck`: 통과
  - `NODE_OPTIONS='--localstorage-file=/tmp/yeon-vitest-localstorage-20260723-2222' pnpm --filter @yeon/web test`: 263개 파일, 1,146개 테스트 통과
  - `pnpm verify:parity`: 14개 검증 개념 통과
- 테스트 참고
  - 기본 Node 26 실행은 기존 cloud-import 테스트가 `localStorage` 파일 미지정으로 3건 실패했다.
  - 동일 스위트에 Node의 localStorage 파일을 지정하자 전부 통과해 이번 변경 실패가 아님을 확인했다.
