# 랜딩 커뮤니티 전경 v5 교체 작업 로그

## 목표

- 제공 이미지를 4번 커뮤니티 카드의 새 투명 전경으로 적용한다.
- 공통 배경은 그대로 유지한다.

## 작업 범위

- `apps/web/public/images/landing/community-frame-break-foreground-v5.webp`
- `apps/web/src/features/landing-home/landing-home.tsx`
- 관련 백로그, 작업 로그, Playwright 스크린샷

## 진행 기록

- 18:19 작업 브랜치를 `origin/main`에서 생성했다.
- 기존 커뮤니티 v4 전경과 새 원본 이미지를 시각 확인했다.
- imagegen 기본 내장 편집 모드에서 원본의 흰 배경만 `#00ff00` 크로마키로 교체했다.
- 공식 `remove_chroma_key.py`의 auto-key, soft-matte, despill 옵션으로 알파 전경을 만들었다.
- 1535×1024 투명 WebP를 품질 88, alpha 품질 100으로 최적화해 154,564바이트로 저장했다.
- 4번 카드가 v5를 참조하도록 이미지 경로와 원본 크기를 변경했다.
- 공통 배경은 수정하지 않았다.

## imagegen 최종 프롬프트 요약

- 배경 추출 용도의 랜딩 커뮤니티 전경.
- 전등 전체, 모니터, 대화 5개와 반응, 공지 패널, 인물, 의자, 키보드, 책상, 접속자 패널을 보존.
- 카메라와 배치를 바꾸지 않고 흰 외부 배경만 균일한 크로마키로 교체.
- 추가 UI, 문구 변경, 잘림, 그림자, 워터마크 금지.
