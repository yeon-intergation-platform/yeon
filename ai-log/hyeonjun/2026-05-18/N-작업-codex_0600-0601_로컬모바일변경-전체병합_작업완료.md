# 로컬 모바일 변경 전체 병합 작업 로그

## 목표

- 로컬에 남아 있던 모바일/카드서비스/dev-all 변경을 origin/main 기준 PR로 병합한다.
- `.tmp/dev-all/*.log`는 Git 추적에서 제거하고 이후 추적되지 않게 한다.

## 진행

- 로컬 변경을 stash로 보존한 뒤 `chore/merge-local-mobile-all` 브랜치를 `origin/main`에서 생성했다.
- stash 적용 중 `scripts/dev-all.mjs` 충돌이 발생했고, upstream의 race-server URL 전달과 로컬 provider env 전달을 모두 유지해 해소했다.
- `.tmp/dev-all/*.log`는 `.gitignore`에 추가하고 `git rm --cached`로 추적 제거했다.

## 검증 예정

- `pnpm --filter @yeon/mobile lint`

- `pnpm --filter @yeon/api-contract typecheck`
- `pnpm --filter @yeon/api-client typecheck`
