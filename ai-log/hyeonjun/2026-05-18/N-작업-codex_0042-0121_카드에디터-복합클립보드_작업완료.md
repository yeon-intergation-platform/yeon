# 카드 에디터 복합 클립보드/이미지 안정화 작업 로그

## 목표

- Yeon 카드 에디터 디자인은 유지한다.
- study-platform-client의 복합 클립보드/이미지 안정화 기능만 선별 이식한다.

## 변경

- 카드 에디터 이미지 업로드 유틸에 파일 header 기반 MIME 보정, HEIC/HEIF → JPEG 변환, data URI/blob 변환을 추가했다.
- 텍스트+이미지 rich HTML 붙여넣기 후 새 이미지 src를 업로드 URL로 치환하는 흐름을 추가했다.
- 이미지 단독 붙여넣기, Clipboard API 이미지, 이미지 URL 붙여넣기, 드롭/버튼 업로드 경로를 같은 검증/업로드 로직으로 통일했다.
- Notion attachment 이미지는 깨진 이미지 대신 재업로드 안내 문구로 치환한다.

## 검증

- `pnpm --filter @yeon/web lint` 통과
- `pnpm --filter @yeon/web typecheck` 통과
- `git diff --check` 통과
- `pnpm --filter @yeon/web build` 통과
