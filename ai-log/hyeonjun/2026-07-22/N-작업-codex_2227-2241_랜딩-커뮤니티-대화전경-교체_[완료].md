## 목표

- 제공 이미지의 대화 UI가 카드 안에서 읽히도록 중앙 안전영역 중심의 투명 전경으로 교체한다.
- 기존 공용 배경은 변경하지 않는다.
- 375px·1040px·1440px Playwright 증거를 남긴다.

## 진행 기록

- 기존 랜딩 전경 PR 브랜치에 최신 `origin/main`을 병합했다.
- built-in imagegen 편집 모드와 공식 크로마키 제거 도구를 사용한다.

## 이미지 제작

- built-in imagegen 편집 모드에서 제공 원본의 인물 정체성, 전등, 모니터, 키보드, 의자와 대화 UI를 유지했다.
- 프롬프트는 대화 말풍선 5개·공지 패널·반응 배지·접속자 패널을 전등과 인물 사이 중앙 안전영역에 모으고, 어떤 요소도 가장자리에 닿지 않도록 지정했다.
- 단색 `#00ff00` 배경으로 생성한 뒤 공식 `remove_chroma_key.py`의 border auto-key·soft matte·despill로 투명화했다.
- 최종 에셋: `apps/web/public/images/landing/community-frame-break-foreground-v4.webp` — 1560×1018.
- 이전 v3 전경은 코드 참조를 v4로 교체한 뒤 제거했다.

## 카드 적용

- 4번 카드만 v4 전경으로 교체했다.
- 카드 전용 폭 `w-[min(150%,31rem)]`와 기존 `-bottom-4` 하단 위치로 번호·제목을 피하면서 전등과 인물 머리가 헤더 하단을 넘게 했다.
- 공용 배경 `baekji-frame-break-background.webp`는 수정하지 않았고 git blob 해시가 `origin/main`과 동일했다.
