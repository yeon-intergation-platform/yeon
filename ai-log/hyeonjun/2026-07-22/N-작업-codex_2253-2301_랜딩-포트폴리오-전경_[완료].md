## 목표

- 제공 이미지의 전등·포트폴리오 모니터·키보드·태블릿·작업 문서를 투명 전경으로 제작한다.
- 9번 카드도 1~8번과 같은 공용 배경 + 투명 전경 구조로 통일한다.
- 375px·1040px·1440px Playwright 증거를 남긴다.

## 진행 기록

- 사용자 변경이 있는 `yeon-2`와 main 미반영 커밋이 있는 `yeon-3`는 건드리지 않고, 깨끗한 `yeon-4`에서 `origin/main` 기반 브랜치를 생성했다.
- built-in imagegen 편집 모드와 공식 크로마키 제거 도구를 사용한다.

## 이미지 제작

- built-in imagegen 편집 모드에서 제공 원본의 전등, 대형 포트폴리오 모니터, 토끼 머그, 키보드, 태블릿, 바인더와 설계 문서를 유지했다.
- 프롬프트는 인물을 추가하지 않고 작업대 장면을 와이드 전경으로 재구성하며, 모든 외곽에 안전 여백을 두고 단색 `#00ff00` 배경만 생성하도록 지정했다.
- 공식 `remove_chroma_key.py`의 border auto-key·soft matte·despill로 투명화했다.
- 최종 에셋: `apps/web/public/images/landing/owner-portfolio-frame-break-foreground-v1.webp` — 1876×852.

## 카드 적용

- 9번 `owner-portfolio` 카드에 투명 전경 메타데이터를 연결했다.
- `w-[min(185%,38rem)]`, `-bottom-6`로 전등과 모니터가 헤더 하단을 넘고 문서가 이미지 하단에 자리하도록 맞췄다.
- 공용 배경은 수정하지 않았고 git blob 해시가 `origin/main`과 동일했다.
