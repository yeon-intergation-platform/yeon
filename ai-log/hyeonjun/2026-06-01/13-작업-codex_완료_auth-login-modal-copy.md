# 로그인 모달 지원 서비스 문구 보정 작업 로그

- 시작: 2026-06-01
- 상태: 완료
- 목표: 로그인 모달에서 지원하지 않는 상담 워크스페이스 언급을 제거하고, 현재 지원 서비스 기준의 공통 계정 안내로 바꾼다.
- 범위: apps/web landing-home login modal, docs/product/backlog, ai-log
- 원칙: 유지보수 대상 서비스는 카드·타자·커뮤니티 3종이라는 프로젝트 정책을 따른다.

## 변경

- 로그인 모달 설명 문구에서 상담 워크스페이스를 제거했다.
- 카드·타자·커뮤니티에서 이어 쓰는 공통 계정 안내로 바꿨다.

## 검증

- `pnpm --filter @yeon/web lint` 통과
- `pnpm --filter @yeon/web typecheck` 통과
- Playwright: 루트 로그인 모달에서 기존 상담 워크스페이스 문구 0개, 새 카드·타자·커뮤니티 문구 1개 확인
