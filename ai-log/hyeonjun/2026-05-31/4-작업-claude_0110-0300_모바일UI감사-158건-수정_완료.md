## 작업 내용

모바일 뷰포트(390×844) 라이브 스크린샷 1~9 화면을 분석해 기록한 UI 문제 **158건**을 전수 검토하고 안전하게 고칠 수 있는 항목을 모두 반영했다. 화면별 비중첩 파일 클러스터로 나눠 병렬 수정 후 클러스터별 granular 커밋(총 30커밋)으로 쌓았다.

### 공통/토큰 레벨 (여러 화면 동시 반영)

- 한글 어절 줄바꿈: `globals.css`에 `word-break: keep-all` + `overflow-wrap: anywhere` 전역 적용 → 음절 단위 고아글자(#1/#6/#30/#40/#52/#53/#71/#89/#105/#114/#118/#152) 해소.
- 헤더 접근성: `ProductHeaderSettingsButton`/`ProductHeaderProfileButton`에 `min-h-11 min-w-11` + aria-label → 타자/카드/커뮤니티/홈 헤더(#7/#28/#43/#58/#65/#94/#100/#101/#120/#147) 일괄 충족. 타자 화면도 `TypingSettingsButton`이 공통 버튼을 상속해 동시 해결.
- 보조 텍스트 대비: `shared-style-constants`·각 화면 본문 역할 회색을 #777/#888/#999 → #666로 통일(#5/#23/#37/#69/#86/#99/#136/#155 부분).

### 화면별 (이번 웨이브 주요 반영)

- 개인 연습(#68/#69/#70/#73/#76): 지표 라벨 한국어화(정확도/진행도/시간), 대비↑, flex-wrap, 레인 라벨 '상대', 현재 글자 밑줄 커서.
