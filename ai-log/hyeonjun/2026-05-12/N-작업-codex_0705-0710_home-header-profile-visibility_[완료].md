# 홈 헤더/내정보/숨김 서비스 정리

## 범위

- 홈 헤더의 서비스 바로가기/서비스 링크 제거
- 상담 기록 워크스페이스 기본 숨김 및 숨김 토글 제공
- 내정보 아이콘 `/profile` 연결 및 내정보 페이지 추가
- 타자/카드/커뮤니티 헤더 브랜드 라벨 정리

## 완료 내용

- 공통 제품 헤더 중앙 서비스 링크를 제거했다.
- 홈의 인증 사용자용 `서비스 바로가기` 버튼을 제거했다.
- 상담 기록 워크스페이스는 기본 숨김으로 두고 `숨김 서비스 보기` 토글로 노출되게 했다.
- 내정보 아이콘 기본 이동 경로를 `/profile`로 바꾸고 내정보 페이지를 추가했다.
- 서비스 헤더 브랜드를 `YEON 타자연습`, `YEON 카드`, `YEON 커뮤니티`로 표시하게 했다.

## 검증

- `pnpm --filter @yeon/web lint` 통과
- `pnpm --filter @yeon/web typecheck` 통과
- `pnpm --filter @yeon/web build` 통과
- `git diff --check` 통과
- 직접 dev 서버는 켜지 않았고, 이미 떠 있던 `http://localhost:3000/` 응답만 확인했다.
