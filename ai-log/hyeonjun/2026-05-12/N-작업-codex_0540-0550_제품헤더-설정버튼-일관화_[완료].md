# 제품 헤더/설정 버튼 일관화 작업 로그

## 목표

- 타자연습, 플래시카드 덱, 커뮤니티, yeon.world 메인 화면 헤더 크기를 플래시카드 덱 기준으로 통일한다.
- 설정 버튼 외형은 typing-service 톱니 버튼 기준으로 통일한다.

## 범위

- apps/web UI 컴포넌트만 수정한다.
- 헤더 내부 링크/문구/서비스별 요소는 유지한다.

## 검증 예정

- git diff --check
- pnpm --filter @yeon/web build

## 구현 요약

- `ProductHeader`로 제품 공통 헤더 프레임을 만들고 높이를 61px로 고정했다.
- 타자연습/카드/커뮤니티/메인 화면의 헤더 프레임만 공통화했고 내부 링크/CTA는 유지했다.
- 설정 버튼은 typing-service 톱니 버튼 모양을 공통 `ProductHeaderSettingsButton`으로 승격하고 카드 설정에도 적용했다.
- 사용자에게 보이는 UI 배치 수정이므로 patch 버전 0.0.3로 bump했다.

## 검증 결과

- `pnpm --filter @yeon/web lint` 통과
- `pnpm --filter @yeon/web build` 통과
- Playwright headless 측정: `/`, `/typing-service`, `/card-service`, `/community` 첫 header/nav 높이 모두 61px
- `git diff --check` 통과
