# 작업 로그: 커뮤니티 채팅 게스트 허용

- 날짜: 2026-05-12
- 작성자: codex
- 상태: 완료
- 목적: 실시간 커뮤니티 채팅에서 전화번호 인증/로그인 게이트 제거

## 작업 내용

- `CommunityChatWidget`에서 전화번호/OTP 인증 UI 제거
- `useCommunityChat`을 공개 피드 기반 게스트 채팅 흐름으로 전환
- 브라우저 localStorage에 익명 게스트 닉네임/비밀번호를 자동 생성해 사용자가 별도 입력 없이 메시지 전송 가능
- 커뮤니티 백로그에 차수 3 정책 추가

## 검증 예정

- 변경 파일 ESLint
- 웹 타입체크 또는 관련 범위 타입 검증
- 빌드 영향 확인
