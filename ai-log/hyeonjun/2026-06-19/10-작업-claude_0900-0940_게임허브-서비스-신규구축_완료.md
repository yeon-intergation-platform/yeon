# 신규 게임 허브 서비스(game.yeon.world) 구축

## 배경

yeon.world에 외부 임베드 게임을 모아 보여주는 게임 허브 추가(참고: tistory 게임 블로그 방식 — iframe + 설명/조작법, 여러 게임 큐레이션). 자체 `public/slime-game`은 미사용. 임베드를 공식 허용하는 플랫폼 게임만.

## 게임 소스 결정

- 후보 검증(Playwright iframe 로드): play2048.co(X-Frame DENY), github.io 오픈소스 게임(임베드 비전제 → 스크립트 ERR_ABORTED), GameDistribution(md5 필요).
- **CrazyGames `/embed/{slug}` 채택** — 임베드 전용, 실제 로드 확인(2048/hextris/snake-io/smash-karts/bullet-force). 무료 임베드 대가로 게임 내 광고 포함(CrazyGames 정책).

## 변경

### 신규 (features/game-service/)

- `game-catalog.ts`: `GameEntry`(slug/title/summary/description/controls/category/provider/embedUrl/orientation) + 카테고리 상수/라벨 + getGameBySlug/getListedGames/getGameSlugs. CrazyGames embed 게임 5종(2048/Hextris/Snake.io/Smash Karts/Bullet Force).
- `game-service-home.tsx`: 허브(카드 그리드, 카테고리 배지). `game-detail.tsx`: 반응형 iframe(sandbox/allow + 전체화면) + 설명/조작법/출처. `index.ts`.

### 신규 라우트 (app/game-service/)

- `page.tsx`(허브 + CollectionPage JSON-LD), `[gameSlug]/page.tsx`(generateStaticParams/generateMetadata + VideoGame JSON-LD + notFound).

### 등록 (기존 기반 재사용)

- `platform-services.ts`: game-service 추가(listedInPlatformHome:true → landing 카드 자동).
- `subdomain-routing.ts`: game.yeon.world → /game-service(proxy 자동 rewrite).
- `product-header.tsx`: CommonServiceKey `game` + "YEON 게임" 라벨 + SERVICE_BASE_PATH(좌상단 네비 자동).
- `seo.ts`: SERVICE_CANONICAL_URLS.game + sitemap(허브 + 게임별).

## 검증

- typecheck/lint exit 0(web), mobile typecheck 0(platform-services 공유 호환).
- 단위 테스트 game-catalog 7케이스 통과.
- Playwright(dev 3100): 허브 desktop/mobile(5개 카드, 헤더 "YEON 게임"), 상세 2048 desktop/mobile(**실제 CrazyGames 게임 iframe 로드 확인** — games.crazygames.com frame). 스크린샷 `ai-log/hyeonjun/2026-06-19/game-service-screenshots/`.

## 리스크 / 후속

- CrazyGames embed는 게임 내 광고/추적(doubleclick·amazon-adsystem 등) 포함 → 운영 privacy/CSP 정책 검토 필요.
- game.yeon.world DNS/Cloudflare Tunnel/Access 등록은 배포 인프라(domain-routing) — 코드 외.
- landing 카드 노출은 listedInPlatformHome:true로 자동(루트 페이지 무거워 스크린샷 미캡처, 메커니즘은 기존 검증).
- 게임 추가는 작동 확인된 CrazyGames slug를 카탈로그에 넣기만 하면 됨. 향후 카테고리 필터/검색/feed 자동화.
